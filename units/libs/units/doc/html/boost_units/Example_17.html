Example 17

(unit_example_17.cpp)

This example demonstrates the abstract physical unit system provided for strictly dimensional analysis calculations. This system explicitly prohibits quantity conversions of any sort to ensure that it cannot be mixed with other unit systems. 

const abstract::length  L;
const abstract::mass    M;
const abstract::time    T;

gives 

L             = [L]
L+L           = [L]
L-L           = [L]
L/L           = dimensionless 
M*(L/T)*(L/T) = [L]^2 [M] [T]^(-2)
M*(L/T)^2     = [L]^2 [M] [T]^(-2)
L^3           = [L]^3
L^(3/2)       = [L]^(3/2)
2vM           = [M]^(1/2)
(3/2)vM       = [M]^(2/3)
