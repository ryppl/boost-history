Macro BOOST_TTI_MTFC_HAS_STATIC_MEMBER

BOOST_TTI_MTFC_HAS_STATIC_MEMBER — Expands to a metafunction class which tests whether a static member data or a static member function with a particular name and type exists.

Synopsis

// In header: <boost/tti/tti.hpp>

BOOST_TTI_MTFC_HAS_STATIC_MEMBER(name)

Description

name = the name of the inner member.

returns = a metafunction class called "boost::tti::mtfc_has_static_member_name" where 'name' is the macro parameter.

The metafunction class's 'apply' metafunction types and return:

T = the enclosing type.
Type = the static member data or static member function type, in the form of a data or function type, in which to look for our 'name'.
returns = 'value' is true if the 'name' exists within the enclosing type, with the appropriate type, otherwise 'value' is false.
