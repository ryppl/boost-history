Macro BOOST_LOCAL_FUNCTION_NAME

BOOST_LOCAL_FUNCTION_NAME — This macro is used to specify the local function name.

Synopsis

// In header: <boost/local/function.hpp>

BOOST_LOCAL_FUNCTION_NAME(name)

Description

This macro must follow the local function body code block { ... } as shown in the BOOST_LOCAL_FUNCTION_PARAMS documentation.

Parameters:

name The name of the local function. The local function name can be prefixed by the keyword inline as in BOOST_LOCAL_FUNCTION_NAME(inline name) to increases the likelihood that the compiler will inline the local function (thus reducing its run-time) but inlined local functions can be passed as template parameters only on C++03 compilers (and not on ISO C++ standard compilers) -- see the Advanced Topics section. The local function name cannot be the name of an operator operator... and it cannot be the same name of another local function declared within the same enclosing scope (i.e., it is not possible to overload local function declarations) -- see the Advanced Topics section.

See: BOOST_LOCAL_FUNCTION_PARAMS, Tutorial section, Advanced Topics section.
