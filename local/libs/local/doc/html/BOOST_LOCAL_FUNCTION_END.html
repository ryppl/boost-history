Macro BOOST_LOCAL_FUNCTION_END

BOOST_LOCAL_FUNCTION_END — This macro ends the definition of a local function.

Synopsis

// In header: <boost/local/function.hpp>

BOOST_LOCAL_FUNCTION_END(local_function_name)

Description

This macro must follow the local function body code { ... } as shown in the BOOST_LOCAL_FUNCTION documentation.

Parameters:

local_function_name The name of the local function as it has been specified by the parenthesized signature passed to BOOST_LOCAL_FUNCTION (otherwise, the compiler will generate an error). See the BOOST_LOCAL_FUNCTION_END_RENAME macro to relax this constraint.

See: BOOST_LOCAL_FUNCTION_END_RENAME, BOOST_LOCAL_FUNCTION, Tutorial section.
