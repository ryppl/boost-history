Macro BOOST_LOCAL_TYPEOF

BOOST_LOCAL_TYPEOF — This macro expands to the fully qualified type of a variable bound to to local functions, local blocks, and local exits.

Synopsis

// In header: <boost/local/typeof.hpp>

BOOST_LOCAL_TYPEOF(bound_variable_name)

Description

The type is fully qualified in that it contains the extra constant and reference qualifiers when they are specified for binding by constant and by reference. For example, if a variable named x of type T is:

  • Bound by value using bind x then BOOST_LOCAL_TYPEOF(x) is T.

  • Bound by constant value using const bind x then BOOST_LOCAL_TYPEOF(x) is const T.

  • Bound by reference using bind& x then BOOST_LOCAL_TYPEOF(x) is T&.

  • Bound by constant reference using const bind& x then BOOST_LOCAL_TYPEOF(x) is const T&.

Within local functions, local blocks, and local exits, this macro can be used to deduce the bound variable types to declare local variables, check concepts (using Boost.ConceptCheck), etc (see Advanced Topics section).

Parameters:

bound_variable_name The name of the bound variable for which the type is being deduced.

See: BOOST_LOCAL_FUNCTION_PARAMS, BOOST_LOCAL_BLOCK, BOOST_LOCAL_EXIT, Advanced Topics section.
