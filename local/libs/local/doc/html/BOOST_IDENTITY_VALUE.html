Macro BOOST_IDENTITY_VALUE

BOOST_IDENTITY_VALUE — This macro wraps the specified value expression within extra parenthesis so the value can be passed as a single macro parameter even if it contains commas.

Synopsis

// In header: <boost/utility/identity.hpp>

BOOST_IDENTITY_VALUE(parenthesized_value)

Description

This macro expands to an expression that can be passed as a single macro parameter even if it contains commas and that evaluates to the specified value at run-time (see the {Advanced_Topics, Advanced Topics} section).

For example BOOST_IDENTITY_VALUE((key_sizeof<int, double>::value)) can be passed as a single macro parameter when instead key_sizeof<int, double>::value cannot (because it contains a comma not wrapped by round parenthesis so it will be interpreted as two separate macro parameters by the preprocessor).

Parameters:

parenthesize_value The value expression to be passed as macro parameter wrapped by a single set of round parenthesis (...). This value expression can contain an arbitrary number of commas.

See: Advanced Topics section.
