Class template overload

boost::local::function::overload — Functor to overload local functions and other functors.

Synopsis

// In header: <boost/local/function/overload.hpp>

template<typename F0, typename F1, ... > 
class overload {
public:
  // construct/copy/destruct
  overload(const boost::function< F0 > &, const boost::function< F1 > &, ...);

  // public member functions
  result_type< F0 > operator()(arg0_type< F0 >, arg1_type< F0 >, ...) const;
  result_type< F1 > operator()(arg0_type< F1 >, arg1_type< F1 >, ...) const;
};

Description

This functor aggregates together calls to functions of all the specified function types F0, F1, etc. Each function type must be specified following the Boost.Function preferred syntax: 

  ResultType (ArgumentType0, ArgumgnetType1, ...)

The maximum number of overloaded function types is specified by the BOOST_LOCAL_CONFIG_OVERLOAD_MAX configuration macro. The maximum number of function parameters for each of the specified function type is specified by the BOOST_LOCAL_CONFIG_OVERLOAD_MAX configuration macro.

See: Advanced Topics section, BOOST_LOCAL_CONFIG_OVERLOAD_MAX, BOOST_LOCAL_CONFIG_OVERLOAD_MAX, Boost.Function.

overload public construct/copy/destruct

  1. overload(const boost::function< F0 > &, const boost::function< F1 > &, ...);
    Construct the overloading functor.

    Any functor that can be converted to a boost::function funcotr can be specified (local functions, function pointers, other functors, etc).

overload public member functions

  1. result_type< F0 > operator()(arg0_type< F0 >, arg1_type< F0 >, ...) const;
    Call operator matching the signature of the function type specified as first.

    This will in turn invoke the call operator of the first functor that was passed to the constructor. 

  2. result_type< F1 > operator()(arg0_type< F1 >, arg1_type< F1 >, ...) const;
    Call operator matching the signature of the function type specified as second.

    This will in turn invoke the call operator of the second functor that was passed to the constructor.
