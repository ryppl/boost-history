Macro BOOST_LOCAL_CONFIG_THIS_PARAM_NAME

BOOST_LOCAL_CONFIG_THIS_PARAM_NAME — The name of the special symbol used to access the bound object this.

Synopsis

// In header: <boost/local/config.hpp>

BOOST_LOCAL_CONFIG_THIS_PARAM_NAME

Description

Note: If programmers leave this macro undefined, the default symbol used is this_.

Warning: Programmers should not define this macro unless it is absolutely necessary (e.g., to avoid name clashes with another library which cannot be changed). Changing the symbol this_ effectively changes the public API of this library.
