Macro BOOST_LOCAL_BLOCK

BOOST_LOCAL_BLOCK — This macro starts the declaration of a local block.

Synopsis

// In header: <boost/local/block.hpp>

BOOST_LOCAL_BLOCK(parenthesized_binding)

Description

This macro must be used within a declarative context, it must be followed by the local block body code { ... } and then by the BOOST_LOCAL_BLOCK_END macro (see the Tutorial section): 

  { // Some declarative context.
      ...

      BOOST_LOCAL_BLOCK(
      parenthesized_binding
      ) exception_specifications_optional {
          ... // Block body.
      } BOOST_LOCAL_BLOCK_END

      ...
  }

Within templates, the special macro BOOST_LOCAL_BLOCK_TPL must be used instead of BOOST_LOCAL_BLOCK.

Note: A return; instruction from within a local block body jumps to the end of the local block body and it does not return the enclosing scope.

Parameters:

parenthesized_binding A Boost.Preprocessor sequence that uses the parenthesized syntax to specify the variables in scope to bind (see the Grammar section).

As usual, exception specifications can be optionally programmed before the body code block (see the Advanced section).

See: Tutorial section, Advanced section, Grammar section, BOOST_LOCAL_BLOCK_TPL, BOOST_LOCAL_BLOCK_END.
