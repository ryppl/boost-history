Macro BOOST_LOCAL_FUNCTION

BOOST_LOCAL_FUNCTION — This macro starts the declaration of a local function.

Synopsis

// In header: <boost/local/function.hpp>

BOOST_LOCAL_FUNCTION(parenthesized_signature)

Description

This macro must be used within a declarative context, it must be followed by the local function body code { ... } and then by either the BOOST_LOCAL_FUNCTION_END or the BOOST_LOCAL_FUNCTION_END_RENAME macro (see the Tutorial and Advanced sections): 

  { // Some declarative context.
      ...

      BOOST_LOCAL_FUNCTION(
      parenthesized_signature
      ) exception_specifications_optional {
          ... // Block body.
      } BOOST_LOCAL_FUNCTION_END(local_function_name)

      ...
  }

As usual, exception specifications can be optionally programmed before the body code block (see the Advanced section).

Within templates, the special macro BOOST_LOCAL_FUNCTION_TPL must be used instead of BOOST_LOCAL_FUNCTION.

Parameters:

parenthesized_signature A Boost.Preprocessor sequence that uses the parenthesized syntax to specify the local function signature and the variables in scope to bind (see the Grammar section).

Note: Local functions cannot be copied but it is possible to obtain a reference to a local function using boost::local::function_ref.

See: Tutorial section, Advanced section, Grammar section, BOOST_LOCAL_FUNCTION_TPL, BOOST_LOCAL_FUNCTION_END, BOOST_LOCAL_FUNCTION_END_RENAME, boost::local::function_ref.
