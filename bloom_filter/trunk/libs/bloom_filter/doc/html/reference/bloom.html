Basic Bloom Filter

template <typename T, size_t Size, class HashFunctions = mpl::vector<boost_hash<T, 3>, boost_hash<T, 37> > > class bloom_filter { // exported typedefs typedef T value_type; typedef T key_type; // constructors bloom_filter(); // data structure metadata query functions static constexpr size_t size() const; static constexpr size_t num_hash_functions() const; double false_positive_rate() const; size_t count() const; // data structures core ops void insert(const T&); bool contains(const T&) const; // union assign/intersect assign bloom_filter& operator|=(const bloom_filter&); bloom_filter& operator&=(const bloom_filter&); }; // union template <typename T, size_t Size, class HashFunctions> bloom_filter<T, Size, HashFunctions> operator|(const bloom_filter<T, Size, HashFunctions>& lhs, const bloom_filter<T, Size, HashFunctions>& rhs); // intersect template <typename T, size_t Size, class HashFunctions> bloom_filter<T, Size, HashFunctions> operator&(const bloom_filter<T, Size, HashFunctions>& lhs, const bloom_filter<T, Size, HashFunctions>& rhs);

The basic Bloom filter data structure. Supports insertion and query.

Template Parameters Reference

T
The type used for all Bloom filter operations.
Size
The size in bits of the underlying std::bitset used by the Bloom filter.
HashFunctions
The set of hash functions used by the Bloom filter. Currently required to be an mpl::vector of Hashers.

bloom_filter()

Description
Constructs a bloom_filter object with all bits set to 0.

size_t size() const

Description
Returns the number of bits used internally by the Bloom filter..
Returns
Value of template parameter Size.

size_t num_hash_functions() const

Description
Returns the number of hash functions used by the Bloom filter.
Returns
mpl::size<HashFunctions>.

double false_positive_rate() const

Description
Returns the current false positive rate based upon the number of hash functions used (k), the number of bits available (m), and the number of bits set (n).
Returns
A double precision value calculated as (1 - e^(k*n/m))^k.

size_t count() const

Description
Returns the number of bits currently set in the Bloom filter.
Returns
std::bitset.count().

void insert(const T&)

Description
Inserts an element into the Bloom filter.
Post-condition
At least 1 bit and at most num_hash_functions() bits of the Bloom filter will have been set.

bool contains(const T&) const

Description
Queries an element in the Bloom filter, checking each hash function value against the set bits.
Returns
True if all hash functions agree with the bitset; false otherwise.
Warning
It may return a false positive if there are hash collisions - this means that depending on the Size of the Bloom filter and the hash functions used, there is a probability that the Bloom filter will return true even though the element was never inserted.

bloom_filter& operator|=(const bloom_filter&)

Description
Performs a union.
Returns
Returns a reference to this.
Post-condition
The number of bits set in this will be greater than or equal to the number of bits set before the operation.

bloom_filter& operator&=(const bloom_filter&)

Description
Performs an intersection.
Returns
Returns a reference to this.
Post-condition
The number of bits set in this will be less than or equal to the number of bits set before the operation.

bloom_filter& operator|(const bloom_filter& lhs, const bloom_filter& rhs)

Description
Performs a union.
Returns
Returns a new, stack-allocated Bloom filter that is the result of performing a union between the lhs and the rhs.
Post-condition
The number of bits set in the new filter will be greater than or equal to the number of bits set in max(lhs.count(), rhs.count()).

bloom_filter& operator&(const bloom_filter& lhs, const bloom_filter& rhs)

Description
Performs an intersection.
Returns
Returns a new, stack-allocated Bloom filter that is the result of performing an intersect between the lhs and the rhs.
Post-condition
The number of bits set in the new filter will be less than or equal to the number of bits set in min(lhs.count(), rhs.count()).
