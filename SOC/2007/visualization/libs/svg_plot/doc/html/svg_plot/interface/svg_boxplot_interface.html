functor

*

A class or that contains a conversion function. You will not have to worry about this, unless you are trying to accomplish stuff like plotting a container of a user-defined class, for example human, that does not have a conversion to double. For example: 

class my_functor
{
  typdef double result_type;

  double operator()(const human& body) const
  {
    return double(body.age());
  }
}

// ...

plot(my_data, "People", functor = my_functor());

boost_default_convert (default) is sufficient in all cases where the data stored in the container can be directly cast to a double.
