The plot() Method

The plot() method is defined using Boost.Parameter. As such, it supports a few extra named parameters, as well as a deduced parameter.

functor

*

A class or that contains a conversion function. You will not have to worry about this, unless you are trying to accomplish stuff like plotting a vector of humans. For example: 

class my_functor
{
	typdef double result_type;

	double operator()(const human& _hum) const
	{
		return double(_hum.age());
	}
}

// snip

plot(my_data, "Lions", functor = my_functor());

boost_default_convert (the default) is sufficient in all cases where the data stored in the container can be directly casted to a double
