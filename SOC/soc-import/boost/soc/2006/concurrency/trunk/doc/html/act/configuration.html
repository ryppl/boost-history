Maximum Number of Call Parameters

Header: #include <boost/config/max_call_params.hpp>

Macro: BOOST_ACT_MAX_PARAMS

The BOOST_ACT_MAX_PARAMS macro holds the maximum number of parameters that an asynchronous function object or queued function object may have when explicitly passing arguments through either queue_function or the constructor of an action. By default, BOOST_ACT_MAX_PARAMS is defined to 10, although this is adjustable by defining BOOST_ACT_MAX_PARAMS to another value in all translation units prior to using any Boost.Act header files.
