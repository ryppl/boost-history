Description and Implementation

The computations of modified spherical Bessel functions are very straightforward. It's just direct evaluations from their definitions.

The function domains are  in(x): (-∞, ∞),  kn(x): (-∞, ∞).

Note

 in(x) is odd function when n is even and even function when n is odd.  kn(x) is neither odd function nor even function.
