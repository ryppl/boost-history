Description and Implementation

Bessel functions of fractional order (i.e. order νis any real number) are much harder to compute than Bessel functions of integer order. Though recurrence relations are the same, the difficulty lies in the initial values.

The function domains are  Jν(x): (0, ∞),  Yν(x): (0, ∞).

To get the initial values, we first normalize ν to [0, ∞) with the help of reflection formulas. Let μ = ν - [ν + 1/2] where [x] is the floor function, then μ is the fractional part of ν such that |μ| <= 1/2 (we need this for convergence later). The idea is to calculate  Jμ(x),  Jμ+1(x),  Yμ(x),  Yμ+1(x) and use them to obtain  Jν(x),  Yν(x).

The algorithm is called Steed's method [3], which needs two continued fractions as well as the Wronskian.

The continued fractions are computed using the modified Lentz's method [13]. Their convergence rates depend on x, therefore we need different strategies for large x and small x.

x > v, CF1 needs O(x) iterations to converge, CF2 converges rapidly

x <= v, CF1 converges rapidly, CF2 fails to converge when x -> 0

Note

Our algorithm calculates  Jμ,  Jμ+1 for large x and  Yμ,  Yμ+1 for small x at the first step, hence the best practice is to compute  Jν and  Yν simultaneously, which is what bessel_jy(v, x, &J, &Y) does exactly.

In the case of small x,  Jν(x) can be calculated directly from the Wronskian without backward recurrence. Some may think symmetrically  Yν(x) can be calculated directly from the Wronskian without forward recurrence for large x. But this is not true because we do not have  Yν+1(x) /  Yν(x) available.

Testing

The test driver calculates relative errors and the following table lists some typical values.

Function float double long double
bessel_jv(x) ~ 10-8 , 10-7  ~ 10-16 , 10-15  ~ 10-19 , 10-18 
bessel_yv(x) ~ 10-8 , 10-7  ~ 10-16 , 10-15  ~ 10-19 , 10-18 

When the function value approaches infinity (this happens when x is small and |ν| is large, |ν| >> x), the relative error grows rapidly. | Jν| ~ 1025 can have relative error 3 or 4 magnitudes higher than that of | Jν| ~ 1.
