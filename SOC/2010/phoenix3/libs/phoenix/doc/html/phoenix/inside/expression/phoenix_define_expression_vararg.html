PHOENIX_DEFINE_EXPRESSION_VARARG

Description

PHOENIX_DEFINE_EXPRESSION_VARARG is a macro that can be used to generate all the necessary boilerplate to create Phoenix Expressions

Synopsis
Semantics
Header 
#include <boost/phoenix/core/expression.hpp>
Example

The example from the previous section could have been rewritten to: 

BOOST_PHOENIX_DEFINE_EXPRESSION(
    (plus)
  , (meta_grammar)               // Cond
    (meta_grammar)               // Do
)
