Evaluator

Description

The concepts of evaluators are the working horses behind phoenix. They are responsible of evaluating the composed expressions.

Various evaluators are thinkable, the main evaluator will be referred to as the default evaluator. It is implemented in terms of the Boost.Proto default context and the various __phoenix_concepts_evaluationpolicies_.
