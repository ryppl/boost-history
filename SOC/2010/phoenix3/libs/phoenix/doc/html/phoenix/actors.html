Actor

The Actor is the main concept behind the library.

Actor

Description

An Actor is way to define a phoenix expression or statement. Actors are TR1 function objects. An actor can accept 0 to PHOENIX_LIMIT arguments.

Note

You can set PHOENIX_LIMIT, the predefined maximum arity an actor can take. By default, PHOENIX_LIMIT is set to 10.
