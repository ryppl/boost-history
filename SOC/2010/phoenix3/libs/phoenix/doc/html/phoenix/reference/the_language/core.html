Nothing
terminal        := value | reference | placeholder

value           := "val" "(" expression ")"

reference       := "ref"  "(" C++ expression ")"
                 | "cref" "(" C++ expression ")"

placeholder     := _1 | _2 | _3 | ... | _N
				| arg1 | arg2 | arg3 | ... | argN

Terminals are the core building blocks. Phoenix, by default distinguishes between 3 types of terminals:

  1. Value Terminals
  2. Reference Terminals
  3. Placeholders

The last two terminal types are handled by the terminal customization points.
