Environment

Description

On an actor function call, bafore evaluating the actors expression, the actor creates an environment. Basically, the environment packages the arguments in a tuple.

Notation

Env

A type fulfilling this concept

env

Object of type Env

N

A MPL Integral Constant

M

A positive Integer index

Expression

Semantics

get_environment_argument<N>(env)
