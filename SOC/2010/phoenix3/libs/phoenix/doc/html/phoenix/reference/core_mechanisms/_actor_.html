actor

Description

actor is main interface to evaluate any Phoenix expression. It is used to wrap the Phoenix Expression Templates and provides various operator() overloads.

Synopsis 
namespace result_of
{
    template <
        typename Expr
      , typename A0 = void
      , typename A1 = void
      ...
      , typename AN = void
      , typename Dummy = void
    >
    struct actor
    {
        typedef unspecified type;
    };
}

template <typename Expr>
struct actor;
Template Parameters

Parameter

Description

Expr

The type of the expression

A0...AN

The type of the supplied arguments

Notation

expr

An instance of a Phoenix actor

a0...a1

Heterogenous values

Expression Semantics

Expression

Semantic

result_of::actor<Expr, A0...AN>::type

The result type of the evaluated expression

boost::result_of<actor<Expr>(A0...AN)>::type

Same as result_of::actor<Expr, A0...AN>::type

expr(a0...aN)

Evaluates the expression

The expression is evaluated using the defined Default Actions.
