bit_width

Description

bit_width is a basic type trait style meta-function which takes a type, and gives you the width of that type in bits.

Header File Information 

#include <boost/integer/bit_width.hpp>

Template Signature

bit_width has the following template signature: 

namespace boost {
template <typename T> struct bit_width;
}

Template Parameters

Parameter

Description

T

A type which you would like the width of.

Interface

Operation

Description

N::value

Returns a value of type std::size_t which is the number of bits within T.

Preconditions

Precondition

Description

Enforcement

You must be able to call sizeof on type T

This is enforced by attempting to take the sizeof T and the compiler telling you you can not.

Examples

Example: 

assert( bit_width<int>::value == 32);
