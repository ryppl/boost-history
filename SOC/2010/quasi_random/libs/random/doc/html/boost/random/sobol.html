Description

class template sobol implements a quasi-random number generator as described in

[Bratley+Fox, TOMS 14, 88 (1988)] and [Antonov+Saleev, USSR Comput. Maths. Math. Phys. 19, 252 (1980)]

In the following documentation X denotes the concrete class of the template sobol returning objects of type IntType, u and v are the values of X.

Some member functions may throw exceptions of type std::overflow_error. This happens when the quasi-random domain is exhausted and the generator cannot produce any more values. The length of the low discrepancy sequence is given by , where digits = std::numeric_limits<IntType>::digits.

sobol public construct/copy/destruct

  1. sobol();

    Effects: Constructs the default Sobol quasi-random number generator.

    Throws: nothing. 

  2. sobol(std::size_t init);

    Effects: Constructs the Sobol quasi-random number generator, equivalent to X u; u.seed(init).

    Throws: overflow_error.

sobol public static functions

  1. static result_type min BOOST_PREVENT_MACRO_SUBSTITUTION() ;

    Returns: Tight lower bound on the set of values returned by operator().

    Throws: nothing. 

  2. static result_type max BOOST_PREVENT_MACRO_SUBSTITUTION() ;

    Returns: Tight upper bound on the set of values returned by operator().

    Throws: nothing. 

  3. static std::size_t dimension() ;

    Returns: The dimension of of the quasi-random domain.

    Throws: nothing.

sobol public member functions

  1. void seed() ;

    Throws: nothing.

    Requirements: *this is mutable.

    Effects: Resets the quasi-random number generator state to the one given by the default construction. Equivalent to u.seed(0). 

  2. void seed(std::size_t init) ;

    Throws: overflow_error.

    Requirements: *this is mutable.

    Effects: Effectively sets the quasi-random number generator state to the init-th vector in the s-dimensional quasi-random domain, where s == X::dimension(). 

         X u, v;
     for(int i = 0; i < N; ++i)
         for( std::size_t j = 0; j < u.dimension(); ++j )
             u();
     v.seed(N);
     assert(u() == v());


  3. result_type operator()() ;

    Requirements: *this is mutable.

    Returns: Returns a successive element of an s-dimensional (s = X::dimension()) vector at each invocation. When all elements are exhausted, X::operator() begins anew with the starting element of a subsequent s-dimensional vector.

    Throws: overflow_error. 

  4. void discard(std::size_t z) ;

    Requirements: *this is mutable.

    Effects: Advances *this state as if z consecutive X::operator() invocations were executed. 

         X u = v;
     for(int i = 0; i < N; ++i)
         u();
     v.discard(N);
     assert(u() == v());

    Throws: overflow_error. 

  5.  BOOST_RANDOM_DETAIL_OSTREAM_OPERATOR(os, sobol, f) ;
    Writes a sobol to a std::ostream. 
  6.  BOOST_RANDOM_DETAIL_ISTREAM_OPERATOR(is, sobol, f) ;
    Reads a niederreiter_base2 from a std::istream. 
  7.  BOOST_RANDOM_DETAIL_EQUALITY_OPERATOR(sobol, x, y) ;
