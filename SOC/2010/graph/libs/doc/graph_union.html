Union and sum are synonyms. The result graph contains vertices and edges present in at least one of the inputs (g1 and g2). In other words, it does the union of vertices and edges from g1 and g2. The graphs must contain name properties for vertices and edges with type size_t and two maps in the graph properties, map<size_t, Vertex> vertices and map<size_t, Edge> edges, mapping names to descriptors. The simple version relies only on vertex naming to identify edges, not needing the vertex names and not receiving the set_edge_label argument.

Parameters

A directed or undirected graph. The graph type must be a model of Vertex List Graph.
IN: const VertexListGraph& g2
A directed or undirected graph. The graph type must be a model of Vertex List Graph.
OUT: MutableGraph& g_out
The union of g1 and g2. The graph type must be a model of Mutable Graph.

Named Parameters

IN: vertex_copy(VertexCopier vc)
This is a function that copies the properties of a vertex in the original graph into the corresponding vertex in the copy.
Default: default_vertex_copier which uses the property tag vertex_all to access a property map from the graph.
IN: vertices_merge(MergeVertices merge_vertices)
This is a function that copies the properties of a vertex in both input graphs into the corresponding vertex in the output.
Default: default_vertices_merge which uses the property tag vertex_all to access a property map from g1, ignoring g2.
IN: set_vertex_label(SetVertexLabel set_vertex_label)
This is a function that sets the name of a vertex.
Default: default_set_vertex_label which sets the name in the vertex property and also the map from names to vertex descriptors in the graph property.
IN: edge_copy(EdgeCopier ec)
This is a function that copies the properties of an edge in the original graph into the corresponding edge in the copy.
Default: default_edge_copier which uses the property tag edge_all to access a property map from the graph.
IN: edges_merge(MergeEdges merge_edges)
This is a function that copies the properties of an edge in both input graphs into the corresponding edge in the output.
Default: default_vertices_merge which uses the property tag vertex_all to access a property map from g1, ignoring g2.
IN: set_edge_label(SetEdgeLabel set_edge_label)
This is a function that sets the name of a edge.
Default: default_set_edge_label which sets the name in the edge property and also the map from names to edge descriptors in the graph property.

Complexity

The time complexity is O(V*R_V+E*R_E), where R_V and R_E are the vertex and edge lookup time.
