Glossary

alive (thread)

a thread is considered alive when not dead.

cleanup decoration

function called at thread exit.

dead (thread)

a thread is considered dead when has not done enough keep alive check points for a given duration.

decoration

Couple of setup/cleanup thread decorating functions.

decorator

Functor wrapper decorating a thread with all the setups and cleanups decorations.

deleter TSSS

specific function used to delete the TSSS.

KA

Keep Alive.

setup decoration

function called before the thread starts.

TSS

Thread Specific Storage.

TSSS

Thread Specific Shared Storage.
