Macro VMD_DATA_SIZE

VMD_DATA_SIZE — Expands to the number of comma-separated variadic macro data arguments.

Synopsis

// In header: <boost/variadic_macro_data/VariadicMacroData.hpp>

VMD_DATA_SIZE(...)

Description

... = variadic macro data.

returns = the number of comma-separated variadic macro data arguments being passed to it.

The value returned can be between 1 and 64.
