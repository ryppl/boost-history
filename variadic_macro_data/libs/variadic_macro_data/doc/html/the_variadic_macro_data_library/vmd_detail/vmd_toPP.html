Convert to Boost PP data types

There are four macros which convert variadic macro data as a whole to each of the four Boost PP data types. These are:

  1. VMD_DATA_TO_PP_TUPLE(...), which converts to a Boost PP tuple.
  2. VMD_DATA_TO_PP_ARRAY(...), which converts to a Boost PP array.
  3. VMD_DATA_TO_PP_LIST(...), which converts to a Boost PP list.
  4. VMD_DATA_TO_PP_SEQ(...), which converts to a Boost PP sequence.
