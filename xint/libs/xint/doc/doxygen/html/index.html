eXtended Integer library.

The Extended Integer (XInt) Library A fast, portable C++ library for multi-precision integer math.

This is the main header file for the library, and the only one that programs using it should need to include.

A C++ library that lets your program handle much, much larger integer numbers than the built-in int, long, or even long long types, and handle them using the same syntax that C and C++ use for the standard integer types.

Completely portable, written entirely in modern C++, with many different types of operating system, compiler, and hardware in mind. It will compile cleanly on many operating systems without any changes, automatically adapting to whatever native integer sizes are available.

It's fast. Speed of execution takes a back seat to portability, so it doesn't include things like assembly-language modules to wring every last CPU cycle out of it -- but it's still pretty darn fast.

Features you need. Modular arithmetic. Bit manipulation functions. Cryptographically-secure random and prime number generation. A friendly and intuitive interface. An option for thread-safe operation.
