Detailed Description

template<typename Collection_Maker, typename Index_Type = std::ptrdiff_t, typename Width_Type = std::size_t>
class boost::index_list::simplex_domain< Collection_Maker, Index_Type, Width_Type >

Domain based on (hyper)planes defined by fixed Manhattan distances.

Assuming the ordering for the indices and defining such that , the effect of reduce() corresponds to the sum . Each possible binomial contribution is tabulated at construction. For reduce() alone, producing a table would not have been necessary. However, expand() would be very slow without it.

Alternatives and comments are welcomed.
