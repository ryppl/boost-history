class configurator

This is core class of library.

API functions

Name Description
configurator Default constructor
add Add option
get_value_of Obtain value of option
load Load options from configuration file
reload Reload options from configuration file
settings Provides access to the configurator_settings class

class single_configurator

This is Singleton-variant of configurator class.

API functions

Name Description
inst Access to single instance of configurator class

class configurator_settings

This is class of Configurator settings.

API functions

Name Description
set_case_sensitivity_for_names Set case sensitivity for names of sections and options
send_warnings_to Set std::ostream for sending possible warnings
set_name_value_separator Set custom name-value separator for options, instead default '='
set_one_line_comment_sign Set custom sign for one-line comments, instead default '//'
set_open_section_edging Set custom edging for section opening
set_close_section_edging Set custom edging for section closing

class option

Name Description

set_default_value Set default value for option
set_necessity Set necessity of option
check_semantic Set checking of some semantic
allow_multi_values Allows existence of multi values for option
