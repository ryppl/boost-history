configurator::add()

signature

template< typename Option >
option& configurator::add();

description

This function adds conceptual option.
Template parameter Option corresponds to type of conceptual option defined with BOOST_CONFIGURATOR_OPTION macro.

parameters

None.

return

Reference to created option corresponding to conceptual option.

exceptions

None.

example

BOOST_CONFIGURATOR_OPTION( Host ) // Conceptual option 'Host' defined.

int main() {
    boost::cf::configurator conf;
    conf.add< Host >();
    // ...
    return 0;
}
