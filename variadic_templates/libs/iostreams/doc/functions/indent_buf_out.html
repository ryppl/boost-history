Overview

An io manipulator which indents outward subsequent output lines written to its std::basic_ostream<Ch>& argument.

Headers

<boost/iostreams/utility/indent_scoped_ostreambuf.hpp>

Reference

Description

The function template indent_buf_out is used on an std::basic_ostream<Ch>& which maybe have been passed to a indent_scoped_ostreambuf<Ch> constructor. If that's the case, then subsequent lines written to this std::basic_ostreamlt;Ch> will be indented outward by the amount specified in the second argument to the indent_scoped_ostreambuf<Ch> constructor.

Synopsis

< class Ch
, class Tr

indent_buf_out(std::basic_ostream<Ch, Tr>& os)

Template Parameters

Ch- The character type
Tr- The traits type

Function Parameters

os- The std::basic_ostream<Ch, Tr> whose subsequent output lines will be indented.

Semantics

The semantics of indent_buf_out depends on whether or not its argument, os, has been passed to a indent_scoped_ostreambuf<Ch>constructor. If so, and that indent_scoped_ostreambuf's destructor has not been called, then subsequent lines output to os will be indented outward by the amount passed as the 2nd parameter to the indent_scoped_ostreambuf<Ch>constructor.
