Description

The class template indent_filter is an OutputFilter which indents each output line by a user-modifiable number of spaces.

Headers

<boost/iostreams/filter/indent.hpp>

Installation

Only the header file is needed.

Reference

Synopsis

namespace boost { namespace iostreams {

#include <boost/iostreams/concepts.hpp>
#include <boost/iostreams/operations.hpp>

template< typename Ch >
class indent_filter 
: public filter< output, Ch>
{   
public:

    static int const width_default=2
      ;
    explicit indent_filter
      ( int width=width_default
      );
    int indentation() const
      ;
    int width() const
      ;
    void indent_by
      ( int amount
      )
      ;
    void indent_out()
      ;
    void indent_in()
      ;
};


} } // End namespace boost::io

Template parameters

Ch- The character type

static int const indent_filter::width_default

The default value for return from width().

indent_filter::indent_filter

indent_filter
  ( int width=width_default
  )

Constructs a indent_filter with given width. The parameters have the following interpretations :

width- The amount by which indentation is changed in calls to indent_in() and indent_out(). If this amount is < 0, then it's reset to 0.

int indent_filter::indentation()const

The current indentation. IOW, each output line is prefixed with indentation() spaces.

int indent_filter::width()const

The absolute amount by which indentation is changed in calls to indent_in() and indent_out() .

indent_filter::indent_by

indent_by
  ( int amount
  )

Changes the indentation by the given amount. If resulting indentation is < 0, resets it to 0.

void indent_filter::indent_in()

Calls indent_by with argument, +width()

void indent_filter::indent_out()

Calls indent_by with argument, -width()
