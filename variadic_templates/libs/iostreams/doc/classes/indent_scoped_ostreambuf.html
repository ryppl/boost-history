Description

The class template indent_scoped_ostreambuf temporarily replaces the std::basic_stream_buf of a std::basic_ostream with a filtering_streambuf<output> which indents each output line by a user-modifiable number of spaces.

indent_scoped_ostreambuf::indent_scoped_ostreambuf

indent_scoped_ostreambuf
  ( std::basic_ostream<Ch>& sout
  , int width=indent_filter<Ch>::width_default
  )

Constructs a indent_scoped_ostreambuf by following these steps:

  1. Saves, in a member variable, the standard stream buffer returned by sout.rdbuf().
  2. Calls sout.rdbuf(indent_buf), where indent_buf is a filtering_streambuf<output> whose sink is the standard stream buffer saved in step 1, and which indents each output line sent to that standard stream buffer by a user-modifiable number of spaces. The user modifications of the number of spaces is done using the following functions:
    1. indent_buf_in for increasing the number of spaces indented, i.e. indenting inward.
    2. indent_buf_out for decreasing the number of spaces indented, i.e. indenting outward.

indent_scoped_ostreambuf::~indent_scoped_ostreambuf

indent_scoped_ostreambuf
  ( void
  )

Restores to the sout argument passed to the constructor, the standard stream buffer saved in step 1 of constructor.
