C++ Concepts Wording and Issues
===============================

This directory contains the sources for the concepts wording and
issues lists. 

Subdirectories:

  issues/       Contains the issues reported against the concepts
                specification, both for the language wording and the
                library wording. Its "issues" subdirectory contains
                XML files for each issue.

  wording/      LaTeX sources for the concepts language wording.


The list of all specification issues for the concepts proposals is
available at
http://www.generic-programming.org/languages/conceptcpp/issues/concepts-active.html.
