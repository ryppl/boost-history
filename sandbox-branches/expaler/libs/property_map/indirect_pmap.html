indirect_pmap

This function object essentially wraps the underlying property map's subscript, [], or get functionality within the () operator. If the property map's value type is bool, then this functor becomes an Adaptable Predicate that can be used in functions that take one as input.

Where Defined

boost/indirect_cmp.hpp

Template Parameters

Parameter Description Default
ReadablePropertyMap The type of the external property map whose key type K is the argument type of this function and whose value type V is the return type of this function.

Models

  • Adaptable Unary Function

Type Requirements

  • ReadablePropertyMap is a model of Readable Property Map
  • K is the type of indirect_pmap's arguments.
  • V is indirect_pmap's return type.

Public Base Classes
