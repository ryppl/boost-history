Case Conversion Algorithms

Overview

Case conversion algorithms are declared in <boost/string_algo/case_conv.hpp> header.
Algorithms tolower and toupper are extensions to std::tolower() and std::toupper() methods for sequences.

Description

tolower and toupper algorithms convert the case of characters in the sequence, character by character. Case conversion is driven by locale settings provided as a parameter.

Function Description
tolower_copy Converts a sequence to lower case. Copy version.
tolower Converts a sequence to lower case. Mutable, in-place version.
toupper_copy Converts a sequence to upper case. Copy version.
toupper Converts a sequence to upper case. Mutable, in-place version.

Synopsis

Iterator based variants. Functions return an output iterator pointing just after last inserted character. 

namespace boost {

    template< typename OutputIteratorT, typename InputIteratorT >
    inline OutputIteratorT tolower_copy(
        OutputIteratorT Output,
        InputIteratorT Begin,
        InputIteratorT End,
        const std::locale& Loc=std::locale() );
    template< typename OutputIteratorT, typename InputIteratorT >
    inline OutputIteratorT toupper_copy(
        OutputIteratorT Output,
        InputIteratorT Begin,
        InputIteratorT End,
        const std::locale& Loc=std::locale() );
}
Sequence version with copy semantics. Returns a new sequence which is a transformed copy of the input sequence
namespace boost {

    template< typename SeqT >
    inline SeqT tolower_copy( const SeqT& Input, const std::locale& Loc=std::locale() );

    template< typename SeqT >
    inline SeqT toupper_copy( const SeqT& Input, const std::locale& Loc=std::locale() );
}
Sequence, in-place version. Returns a reference to the input sequence.
 namespace boost {

    template< typename SeqT >
    inline SeqT& tolower( SeqT& Input, const std::locale& Loc=std::locale() );

    template< typename SeqT >
    inline SeqT& toupper( SeqT& Input, const std::locale& Loc=std::locale() );
}
Parameters:
typename OutputIteratorT An output iterator type
typename InputIteratorT An input iterator delimiting the input
typename SeqT A sequence type (must satisfy the ContainerConcept)
OutputIteratorT Output The output iterator for the result. The result of the operation will be copied to this iterator.
InputIteratorT Begin The beginning of the input sequence
InputIterator End The end of the input sequence
const SeqT& Input The input sequence (const)
SeqT& Input The input sequence (mutable)
const std::locale& Loc The instance of locales which will be used for conversion

Examples

The following example shows basic usage of case conversion algorithms. More examples can be found in the library tests in <libs/string_algo/test/conv_test.cpp>. 

#include <string>
#include <iostream>
#include <iterator>
#include <boost/string_algo/replace.hpp>
#include <boost/string_algo/erase.hpp>

using namespace std;
using namespace boost;

void replace_example()
{  
    cout << "* Replace Example *" << endl << endl;

    string str1("abc___cde___efg");

    cout << "str1 without 6th to 9th character:" <<
        erase_range_copy( 
            str1, 
            str1.begin()+6,
            str1.begin()+9 ) << endl;

    cout << "str1 with 6th to 9th character replaced with '+++': " << 
        replace_range_copy( 
            str1, 
            str1.begin()+6,
            str1.begin()+9,
            string("+++") ) << endl;

    cout << endl;
}
