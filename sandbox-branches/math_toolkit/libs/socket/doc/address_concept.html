Address Concept

The Address concept represents a communication end point.

To enable the user to use the address as a key in containers, etc, the concept refines LessThanComparable.

Name resolution is explicitly excluded from this concept. Rationale: All name lookup and resolution (gethostbyname, inet_addr, et al) is address family specific. Name resolution requires network operations which are prone to failure.

Refinement of

  • DefaultConstructable
  • CopyConstructable
  • Assignable
  • LessThanComparable
  • EqualityComparable
