Class template endian_pack

boost::integer::endian_pack

Synopsis

// In header: <boost/integer/endian/endian_pack.hpp>

template<typename E, typename T, std::size_t n_bits = sizeof(T)*8, 
         alignment A = alignment::aligned> 
class endian_pack {
public:
  // types
  typedef E endian_type;
  typedef T value_type; 

  // construct/copy/destruct
  endian_pack();
  template<typename T2> explicit endian_pack(T2);
  endian_pack& operator=(T);

  // public member functions
  operator T() const;
  const char * data() const;

  // public data members
  constexpr std::size_t width;
  constexpr alignment alignment_value;
};

Description

An endian_pack is a byte-holder with user-specified endianness, value type, size, and alignment.

Requires:

  • E is one of endianness::big or endianness::little.

  • T must be a POD with value semantics.

  • nbits is a multiple of 8

  • If A is alignment::aligned then nbits must be equal to 8*sizeof(T)

endian_pack public construct/copy/destruct

  1. endian_pack();

    Effects: Constructs an object of type endian_pack<E,T,n_bits,A>

    Note(s): if BOOST_ENDIAN_FORCE_PODNESS is defined && C++0x POD's are not available then this constructor will not be present 

  2. template<typename T2> explicit endian_pack(T2 val);

    Effects:: Constructs an object of type endian_pack<E,T,n_bits,A>. x==v, where x is the constructed object.

    Note(s): if BOOST_ENDIAN_FORCE_PODNESS is defined && C++0x POD's are not available then this constructor will not be present 

  3. endian_pack& operator=(T val);

    value_type(*this)==val.

    Returns: *this.

endian_pack public member functions

  1. operator T() const;

    Returns: The current value stored in *this, converted to value_type. 

  2. const char * data() const;

    Returns: The pointer to current value stored in *this, converted to const char*.
