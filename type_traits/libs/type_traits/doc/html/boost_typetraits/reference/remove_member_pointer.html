remove_member_pointer 

template <class T>
struct remove_member_pointer
{
   typedef see-below type;
};

type: The same type as T, but with any pointer-to-member modifier removed.

C++ Standard Reference: 8.3.3.

Compiler Compatibility: Requires that the compiler supports partial specialization of class-templates.

Header: #include <boost/type_traits/remove_member_pointer.hpp> or #include <boost/type_traits.hpp>

Table 1.24. Examples

Expression

Result Type

remove_member_pointer<int>::type

int

remove_member_pointer<int const*>::type

int (short)

remove_member_pointer<int (myclass::*)(short)const>::type
