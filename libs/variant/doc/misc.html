Compatibility

To date the code and test harness have been compiled and tested successfully using Microsoft Visual C++ 7.0 and GNU g++ 2.95.

Credits

The boost::variant library has been inspired by these sources:

[1] Andrei Alexandrescu: "Generic<Programming>: Discriminated Unions" Part 1, Part 2, Part 3, C/C++ Users Journal, 2002

[2] Doug Gregor, BOOST_USER: variant, Boost Wiki paper.

[3] Kevlin Henney: boost::any.

[4] Andrei Alexandrescu: Modern C++ Design, Addison-Wesley, C++ In-Depth series, 2001.

[5] Herb Sutter: Exceptional C++: 47 Engineering Puzzles, Programming Problems, and Solutions, Addison-Wesley, C++ In-Depth series, 2000.

[6] Anthony Williams, suggested the method by which boost::variant guarantees strong exception-safety for stack-stored object.

[7] Eric Friedman, Variant (aka Discriminated Union).

[8] Various boost members, Proposal --- A type-safe union, Boost public discussion.

Dave Abrahams, Brad King, Giovanni Bajo, Fernando Cacciola, and others provided helpful feedback and suggestions to refine the semantics, interface and implementation of the library.
