boost::variant

Introduction

The boost::variant template class offers a simple, type-safe solution for manipulating an object from an inhomogeneous set of  types in a uniform manner. Once a variant object is instantiated, it is very simple for the rest of the code to handle an apparently unrelated set of types in a polymorphic way. This reduces code duplications and enhances maintainability.

Features of boost::variant:

  • Full value semantics, including adherence to standard overload resolution rules
  • Strong exception-safety guarantee
  • Compile-time type-safe value visitation
  • Run-time checked explicit value extraction
  • Support for incomplete types and recursive variant types
  • Efficient stack-based storage solution
