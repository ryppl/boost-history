boost::variant

Introduction

boost::varaint offers a type-safe solution for manipulating several distinguished types in a unifrom manner. Once a variant object is instantiated, it becmoes very simple for the rest of the code to handle, an apparently unrelated set of types, in a polymorphic way. This reduces code duplications and enhances maintainability.
Boost::variant's main features are:
  • Full value semantics
  • A type-safe, Visitor-based facility for extracting the held value
  • Overload resolution rules used for initialization/assignment/visitation
  • High degree of safety thru tight compile-time checking
  • Support for incomplete types and recursive variant types
