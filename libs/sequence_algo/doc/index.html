Introduction

The purpose of this library is to extend and enhance the capabilities of the standard library. It is very understandable that the standard cannot include algorithms for everything, but many C++ users find a need for more algorithms than are currently present. An important part is not only the new algorithms, but also overloaded versions of existing algortihms that works on containers. This can be thought of merely as syntactic sugar, but it opens up for safer and better functional style programming.

Overview

Here is a list of the modules that make up the library:

  • Standard Container Algortihms: overloaded versions of the standard library algorithms that will work with a variety of containers.
  • Min/Max Algorithms: find the minimum and maximum value in one go.
  • Combinatorial Algorithms: extensions that complement std::next/prev_permutation().
  • Longest Common Subsequence: construct a sequence from two source sequences.
  • Miscellaneous: predicates like none() and all() and more.

Everything can be included by using one big header sequence_algo.hpp. Alternatively, each library documents their own header policy for more fine-grained control. Please note that that everything is put in namespace boost::algorithm.
