Class template factory

boost::extensions::factory —

Synopsis

template<typename T, typename Params...> 
class factory {
public:
  // construct/copy/destruct
  factory();
  factory(factory< T > const &);
  factory& operator=(factory< T > const &);

  // public member functions
  template<typename D> void set() ;
  bool is_valid() ;
  T * create(Params...) ;
};

Description

This class is a function object that returns new instances of type T, using factories that take parameters described in the variable length list Params...

factory public construct/copy/destruct

  1. factory();
  2. factory(factory< T > const & first);
  3. factory& operator=(factory< T > const & first);

factory public member functions

  1. template<typename D> void set() ;

    This sets the factory function to the constructor for type D. Example: factory<Base, int, int> f; f.set<Derived>(); 

  2. bool is_valid() ;

    Until set is called, a factory cannot be used. This function can be used to determine if set has been called.

    Requires:

    None.

    Postconditions:

    None.

    Returns:

    True if the factory is initialized (ie, set has been called). 

  3. T * create(Params...) ;

    Returns an instance of T (but does NOT retain ownership of the instance).

    Requires:

    is_valid() == true.

    Postconditions:

    None.

    Returns:

    An instance of T.
