Description

basic_factory_map public construct/copy/destruct

  1. ~basic_factory_map();

basic_factory_map public member functions

  1. template<typename Interface, typename Info, typename Params...> 
  std::map< Info, factory< Interface, Params...> > & get() ;

    This returns a map of the given type of factories. It can return an empty map if no such factories are found.

    Returns:

    A map of the requested factory type.

    		 
  2. template<typename Interface, typename Info, typename Params...> 
  operator std::map< Info, factory< Interface, Params...> > &() ;

    A conversion operator for convenience in calling functions that take a map of factories. This is identical to the get() function.
