Class template basic_parameter_map

boost::reflections::basic_parameter_map — A container for generic_parameters.

Synopsis

template<typename Info = std::string, 
         typename TypeInfo = extensions::default_type_info> 
class basic_parameter_map {
public:
  // types
  typedef std::multimap< Info, generic_parameter< TypeInfo > * > map_type;
  typedef std::multimap< Info, generic_parameter< TypeInfo > * > map_type;

  // construct/copy/destruct
  ~basic_parameter_map();
  ~basic_parameter_map();

  // public member functions
  template<typename D>

    std::vector< const generic_parameter< TypeInfo > * > get(Info) const;
  template<typename D> 
    std::vector< generic_parameter< TypeInfo > * > get(const Info &) ;
  template<typename D> bool has(const Info &) const;
  template<typename D> 
    generic_parameter< TypeInfo > * get_first(const Info &) ;
};

Description

The basic_parameter_map is designed to hold arbitrary data indexed by the value of the Info parameter, and can return all items that match a certain type and Info parameter, ie all integers called "foobar".

basic_parameter_map public construct/copy/destruct

  1. ~basic_parameter_map();

    The destructor deletes all of the parameters that it references.
