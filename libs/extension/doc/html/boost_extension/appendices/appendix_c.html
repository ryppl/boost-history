Appendix C - OS X Shared Libraries

OS X has two kinds of shared libraries, dylibs and bundles. Either one will work with the shared_library class. Be aware of the following limitations however:

  • A bundle cannot be linked to statically (no inheriting from classes defined in bundles).
  • A dylib cannot be unloaded.
