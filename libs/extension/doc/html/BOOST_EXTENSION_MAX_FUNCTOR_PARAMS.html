Macro BOOST_EXTENSION_MAX_FUNCTOR_PARAMS

BOOST_EXTENSION_MAX_FUNCTOR_PARAMS —

Synopsis

BOOST_EXTENSION_MAX_FUNCTOR_PARAMS

Description

This determines the maximum number of parameters that a constructor or exported shared library function can have. 10 is the same default as Boost.Function.
