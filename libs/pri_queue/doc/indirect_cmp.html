indirect_cmp

This function object adaptor imposes an external ordering on the elements it compares, an ordering other than that which is intrinsic to the elements themselves. The ability to impose external ordering is vital for graph operations such as Dijkstra's shortest paths algorithm, which orders vertices according to their distance from the source (and the vertices themselves have no intrinsic ordering).

Where Defined

boost/indirect_cmp.hpp

Template Parameters

Parameter Description Default
ReadablePropertyMap The type of the external property map whose key type K is the type of the elements to be compared and whose value type V is the type of criterion by which the elements are compared.
Compare The type of the function object used to compare the values stored in the external property map.

Models

  • Binary Predicate
  • Adaptable Binary Predicate

Type Requirements

  • ReadablePropertyMap is a model of Readable Property Map
  • K is the type of indirect_cmp's arguments.
  • V is LessThan Comparable.
