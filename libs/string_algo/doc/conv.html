Case Conversion Algorithms

Overview

Case conversion algorithms are included in <boost/string_algo/conv.hpp> header ( in boost namespace ).
Algorithms tolower and toupper are extensions to std::tolower() and std::toupper() methods for sequences.

Description

tolower and toupper algorithms convert the case of characters in the sequence, charater by character. Case conversion is driven by locale settings provided as a parameter.

Function Description Example
toupper_copy Converts a sequence to upper case. Copy version. toupper_copy(ostream_iterator<char>(cout), Begin, End );
str=toupper_copy(str);
toupper Convers a sequence to upper case. Mutable, in-place version toupper(str);
tolower_copy Converts a sequence to lower case. Copy version. tolower_copy(ostream_iterator<char>(cout), Begin, End );
str=tolower_copy(str);
tolower Convers a sequence to lower case. Mutable, in-place version tolower(str);

Synopsis

Iterator based variants. Funtions return an output iterator pointing just after last inserted character. 

    template< typename InputIteratorT, typename OutputIteratorT >
    inline OutputIteratorT tolower_copy(
        OutputIteratorT Output,
        InputIteratorT Begin,
        InputIteratorT End,
        const std::locale& Loc=std::locale() );
    template< typename InputIteratorT, typename OutputIteratorT >
    inline OutputIteratorT toupper_copy(
        OutputIteratorT Output,
        InputIteratorT Begin,
        InputIteratorT End,
        const std::locale& Loc=std::locale() );
Sequence version with copy semantics. Returns a new sequence which is a transformed copy of the input sequence
    template< typename SeqT >
    inline SeqT tolower_copy( const SeqT& Input, const std::locale& Loc=std::locale() );

    template< typename SeqT >
    inline SeqT toupper_copy( const SeqT& Input, const std::locale& Loc=std::locale() );
Sequence, in-place version. Returns a reference to the input sequence.
    template< typename SeqT >
    inline SeqT& tolower( SeqT& Input, const std::locale& Loc=std::locale() );

    template< typename SeqT >
    inline SeqT& toupper( SeqT& Input, const std::locale& Loc=std::locale() );
Parameters:
typename InputIteratorT A forward iterator type
typename OutputIteratorT An output iterator type
typename SeqT A sequence type
OutputIteratorT Output Output iterator for the result. The result of the operation will be sent to this iterator.
InputIteratorT Begin Begin of an input sequence
InputIterator End End of an input sequence
const SeqT& Input An input sequence (const)
SeqT& Input An input sequence (mutable)
const std::locale& Loc An instance of locales which will be used for conversion

Examples

Following example shows basic usage of case conversion algorithms. More examples can be found in the library tests in <libs/string_algo/test/conv_test.cpp>. 

#include <string>
#include <vector>
#include <iostream>
#include <iterator>
#include <boost/string_algo/conv.hpp>

using namespace std;
using namespace boost;

void conv_example()
{  
    cout << "* Case Conversion Example *" << endl << endl;

    string str1("AbCdEfG");
    vector<char> vec1( str1.begin(), str1.end() );

    cout << "lower-cased copy of vec1: ";
    tolower_copy( ostream_iterator<char>(cout), vec1.begin(), vec1.end() );
    cout << endl;

    cout << "upper-cased copy of str1: " << toupper_copy( str1 ) << endl;

    tolower( str1 );
    cout << "lower-cased str1: " << str1 << endl;

    cout << endl;
}
