Overview

Each replace algorithm replaces an occurence of substring in the input sequence. Selection of replaced substring and the substitutuion for it depends on algorithm. Algorithms

Replace Algorithms

Description

Regular variants are included in <boost/string_algo/replace.hpp> header and regex variants are included in <boost/string_algo/regex.hpp> header ( both in boost namespace ).

 

Function Description Example
replace_range Replace the specified range with the format string replace_range_copy(str1, Begin, End, string("23"));
replace_range(Input, Begin, End, string("23"));
replace_first Replace the first occurence of the search string with the format string replace_first_copy(str1, string("<>"), string("23"));
replace_first(Input, string("<>"), string("23"));
replace_last Replace the last occurence of the search string with the format string replace_last_copy(str1, string("<>"), string("23"));
replace_last(Input, string("<>"), string("23"));
replace_nth Replace the n-th occurence of the search string with the format string.
Position is 0-based.		 replace_nth_copy(str1, string("<>"), string("23"), 2);
replace_nth(Input, string("<>"), string("23"), 2);
replace_all Replace all occurences of the search string with the format string replace_all_copy(str1, string("<>"), string("23"));
replace_all(Input, string("<>"), string("23"));
replace_regex Format a substring matching given regular expression with the specified format replace_regex_copy(str1, regex("([A-Z]).*xxx"), string("$1-yyy"));
replace_regex(Input, str1, regex("([A-Z]).*xxx"), string("$1-yyy"));
replace_all_regex Format all substrings matching given regular expression with the specified format replace_all_regex_copy(str1, regex("([A-Z]).*xxx"), string("$1-yyy"));
replace_all_regex(Input, str1, regex("([A-Z]).*xxx"), string("$1-yyy"));

Remark on find_regex

find_regex returns regex_search_result instead of  iterator_range. regex_search_result is derived from iterator_range, but in addition it contains a match_results from regex_search (defined in Regex++) algorithm. It is defined in <boost/string_algo/detail/find_regex.hpp> header.

Examples

Following example shows basic usage of predicate algorithms. More examples can be found in the library tests in <libs/string_algo/test/predicate_test.cpp>. 

#include <string>
#include <iostream>
#include <boost/string_algo/conv.hpp>
#include <boost/string_algo/find.hpp>

using namespace std;
using namespace boost;

void find_example()
{  
    cout << "* Find Example *" << endl << endl;

    string str1("abc___cde___efg");

    // find "cde" substring
    iterator_range<string::iterator> range=find_first( str1, string("cde") );

    // convert a substring to upper case 
    // note that iterator range can be directly passed to the algorithm
    toupper( range );

    cout << "str1 with upper-cased part matching cde: " << str1 << endl;
    cout << endl;
}
