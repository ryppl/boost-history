Overview

String Algorithms Library contains a set of algorithms for replace and erase operations on sequences. Each algorithm finds a range in the input sequence matching given criteria and injects a replacement sequence into this range. Replacement sequence does not have to be constant. Implementation allows generic formatting used for example in variants using regular expressions.

Description

Regular variants are included in <boost/string_algo/replace.hpp> header and regex variants are included in <boost/string_algo/regex.hpp> header ( both in boost namespace ).

Implementation considerations

Replace algoritms are implemented using layered structure. Lower layer implements generic substitution of a range in an input sequence with another sequence. This layer take an input in form of find and format functor. These two functors define what is to be replaced and what will be the replacement. Upper layer functions are just wrapping calls to lower layer provided with specific functors.
