replace_nth/erase_nth

Description

replace_nth is included in <boost/string_algo/replace.hpp> header erase_nth is included <boost/string_algo/erase.hpp> header ( both in boost namespace ).

Algorithm finds a n-th occurrence of the search string within the input sequence. replace_nth substitutes this occurrence with specified format sequence. erase_nth removes this occurrence from the input sequence.

Synopsis

Algorithm is provided in 4 subvariants. Three variants with copy semantics ( all have suffix _copy ) and one in-place variant.

Copy variant with iterator based input. Result is copied into an output iterator.

Examples

Following example shows basic usage of replace_range algorithms. More examples can be found in the library tests in <libs/string_algo/test/replace_test.cpp> and in <libs/string_algo/example/replace_example.cpp>. 

#include <string>
#include <iostream>
#include <iterator>
#include <boost/string_algo/replace.hpp>
#include <boost/string_algo/erase.hpp>

using namespace std;
using namespace boost;

void replace_example()
{  
    cout << "* Replace Example *" << endl << endl;

    string str1("abc___cde___efg");

    // replace third and 5th occurrence of _ in str1
    // note that nth argument is 0-based
    replace_nth( str1, string("_"), 4, string("+") );
    replace_nth( str1, string("_"), 2, string("+") );

    cout << "str1 with third and 5th occurrence of _ replace: " << str1 << endl;
    cout << endl;
};
