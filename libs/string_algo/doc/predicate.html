Predicates

Overview

Predicates are included in <boost/string_algo/predicate.hpp> header ( in boost namespace ).
Provided algorithms are resolving if a substring is contained in the input string under various conditions. This includes suffix and prefix check, equality check and containment check.

Description

 Following table shows the summary of all predicates. Each algorithm returns a bool value representing a success of condition check.

Function Description Example
starts_with Checks if the substring is a prefix of the input. bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("123") );
ends_with Checks if the substring is a suffix of the input. bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("abc") );
contains Checks if the input contains an instance of substring. bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("23a") );
equals Checks if both operands are equal.
( Note: this is two-directional equivalent of std::equal algorithm)		 bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("123abc") );

Sequence variants. 

    template< typename SeqT1, typename SeqT2 >
    inline bool starts_with( const SeqT1& Input, const SeqT2& Substr );
    template< typename SeqT1, typename SeqT2 >
    inline bool ends_with( const SeqT1& Input, const SeqT2& Substr );

    template< typename SeqT1, typename SeqT2 >
    inline bool equals( const SeqT1& Input, const SeqT2& Substr );

Parameters:

typename InputIteratorT A forward iterator type
typename SubIteratorT A forward iterator type
typename SeqT1, typename SeqT2 A sequence types
InputIteratorT Begin Begin of an input sequence
InputIteratorT End End of an input sequence
SubIteratorT SubBegin Begin of a substring sequence
SubIteratorT SubEnd End of a substring sequence
const Seq1T& Input An input sequence
const Seq2T& Substr An substring sequence

Examples

Following example shows basic usage of predicate algorithms. More examples can be found in the library tests in <libs/string_algo/test/predicate_test.cpp>. 

#include <string>
#include <iostream>
#include <boost/string_algo/predicate.hpp>

using namespace std;
using namespace boost;

void predicate_example()
{
    cout << "* Predicate Example *" << endl << endl;

    string str1("123xxx321");
    string str2("abc");

    cout << "str1 starts with \"123\": " << 
        (starts_with( str1, string("123") )?"true":"false") << endl; 
    cout << "str1 ends with \"123\": " << 
        (ends_with( str1, string("123") )?"true":"false") << endl; 

    cout << "str1 contains \"xxx\": " << 
        (contains( str1, string("xxx") )?"true":"false") << endl; 

    // Note: equals have to be explicitly qualified, 
    // otherwise it conflicts with std::equals
    cout << "str2 equals \"abc\": " << 
        (boost::equals( str2, string("abc") )?"true":"false") << endl; 

    cout << endl;
}
