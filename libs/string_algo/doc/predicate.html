Predicates

Overview

Predicates are included in <boost/string_algo/predicate.hpp> header ( in boost namespace ).
Provided algorithms are resolving if a substring is contained in the input string under various contidions. This includes suffix and prefix chech, equality check and containment check.

Description

 Following table shows the summary of all predicates. Each algoritm returns a bool value representing a success of condition check.

Function Description Example
starts_with Checks if the subsring is a prefix of the input. bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("123") );
ends_with Checks if the subsring is a suffix of the input. bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("abc") );
contains Checks if the input contains an instance of substring. bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("23a") );
equals Checks if both operands are equal.
( Note: this is two-directional equivalent od std::equal algorithm)		 bool b=starts_with( begin, end, subbegin, subend );
bool b=starts_with( string("123abc"), string("123abc") );
