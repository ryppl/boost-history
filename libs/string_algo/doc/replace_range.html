String Algorithm Library 

replace_range/erase_range
  • Description
  • Synopsis    
  • Examples

Description

replace_range is included in <boost/string_algo/replace.hpp> header erase_range is included <boost/string_algo/erase.hpp> header ( both in boost namespace ).

Performs a substitution of the specified range in the input sequence. replace_range substitutes this range with specified format sequence. erase_range removes this sequence.

Synopsis

Every algorithm is provided in 4 subvariants. Three variants with copy semantics ( all have suffix _copy ) and one in-place variant.

Copy variant with iterator based input. Result is copied into an output iterator. 

    template< 
        typename InputIteratorT, 
        typename FormatIteratorT,
        typename OutputIteratorT >
    inline OutputIteratorT replace_range_copy(
        OutputIteratorT Output,
        InputIteratorT Begin,
        InputIteratorT End,
        InputIteratorT SearchBegin,
        InputIteratorT SearchEnd,
        FormatIteratorT FormatBegin,
        FormatIteratorT FormatEnd );

   template< 
        typename InputIteratorT, 
        typename OutputIteratorT >
    inline OutputIteratorT erase_range_copy(
        OutputIteratorT Output,
        InputIteratorT Begin,
        InputIteratorT End,
        InputIteratorT SearchBegin,
        InputIteratorT SearchEnd );

Copy variants with a sequence based input. Result is copied into an output iterator. 

    template< 
        typename InputT, 
        typename FormatT,
        typename OutputIteratorT >
    inline OutputIteratorT replace_range_copy(
        OutputIteratorT Output,
        const InputT& Input,
	typename InputT::const_iterator SearchBegin,
	typename InputT::const_iterator SearchEnd,
        const FormatT& Format );

    template< typename InputT, typename OutputIteratorT >
    inline OutputIteratorT erase_range_copy(
        OutputIteratorT Output,
        const InputT& Input,
	typename InputT::const_iterator SearchBegin,
	typename InputT::const_iterator SearchEnd );

Copy variants with a sequence based input. Result is a new sequence.

In-place variant. Returns the reference to the modified input. 

    template< typename InputT, typename FormatT >

        InputT& Input,

Examples

Following example shows basic usage of replace_range algorithms. More examples can be found in the library tests in <libs/string_algo/test/replace_test.cpp> and in <libs/string_algo/example/replace_example.cpp>. 

#include <string>
#include <iostream>
#include <iterator>
#include <boost/string_algo/replace.hpp>
#include <boost/string_algo/erase.hpp>

using namespace std;
using namespace boost;

void replace_example()
{  
    cout << "* Replace Example *" << endl << endl;

    string str1("abc___cde___efg");

    cout << "str1 without 6th to 9th character:" <<
        erase_range_copy( 
            str1, 
            str1.begin()+6,
            str1.begin()+9 ) << endl;

    cout << "str1 with 6th to 9th character replaced with '+++': " << 
        replace_range_copy( 
            str1, 
            str1.begin()+6,
            str1.begin()+9,
            string("+++") ) << endl;

    cout << endl;
};
