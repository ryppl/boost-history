replace_range/erase_range

Description

replace_range is declared in <boost/string_algo/replace.hpp> header erase_range is declared <boost/string_algo/erase.hpp> header.

Performs a substitution of the specified range in the input sequence. replace_range substitutes this range with the specified format sequence. erase_range removes this sequence.

Function Description
replace_range_copy Replace the specified range with the format string. Copy version.
replace_range Replace the specified range with the format string. In-place version.
erase_range_copy Remove the specified range from the sequence. Copy version.
erase_range Remove the specified range from the sequence. In-place version.

Synopsis

The algorithm is provided in 4 subvariants. Three variants with copy semantics ( all have suffix _copy ) and one in-place variant.

Copy variant with iterator based input. Result is copied into an output iterator.

Examples

Following example shows basic usage of replace_range algorithms. More examples can be found in the library tests in <libs/string_algo/test/replace_test.cpp> and in <libs/string_algo/example/replace_example.cpp>. 

#include <string>
#include <iostream>
#include <iterator>
#include <boost/string_algo/replace.hpp>
#include <boost/string_algo/erase.hpp>

using namespace std;
using namespace boost;

void replace_example()
{  
    cout << "* Replace Example *" << endl << endl;

    string str1("abc___cde___efg");

    cout << "str1 without 6th to 9th character:" <<
        erase_range_copy( 
            str1, 
            str1.begin()+6,
            str1.begin()+9 ) << endl;

    cout << "str1 with 6th to 9th character replaced with '+++': " << 
        replace_range_copy( 
            str1, 
            str1.begin()+6,
            str1.begin()+9,
            string("+++") ) << endl;

    cout << endl;
};
