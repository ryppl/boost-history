replace_last/erase_last

Description

replace_last is declared in <boost/string_algo/replace.hpp> header erase_last is declared <boost/string_algo/erase.hpp> header.

The algorithm finds a last occurrence of the search string within the input sequence. replace_last substitutes this occurrence with the specified format sequence. erase_last removes this occurrence from the input sequence.

Function Description
replace_last_copy Replace the last occurrence of the search string with the format string. Copy version.
replace_last Replace the last occurrence of the search string with the format string. In-place version.
erase_last_copy Remove the last occurrence from the sequence. Copy version.
erase_last Remove the last occurrence from the sequence. In-place version.

Synopsis

The algorithm is provided in 4 subvariants. Three variants with copy semantics ( all have suffix _copy ) and one in-place variant.

Copy variant with iterator based input. Result is copied into an output iterator.

Examples

Following example shows basic usage of replace_range algorithms. More examples can be found in the library tests in <libs/string_algo/test/replace_test.cpp> and in <libs/string_algo/example/replace_example.cpp>. 

#include <string>
#include <iostream>
#include <iterator>
#include <boost/string_algo/replace.hpp>
#include <boost/string_algo/erase.hpp>

using namespace std;
using namespace boost;

void replace_example()
{  
    cout << "* Replace Example *" << endl << endl;

    string str1("abc___cde___efg");

    cout << "str1 with 'cde' replaced with 'XYZ': ";
    replace_last_copy( 
        ostream_iterator<char>(cout),
        str1,
        string("cde"),
        string("XYZ") );

    cout << endl;

    cout << "str1 with last '___' replaced with '+++': " << 
        replace_last_copy( 
            str1, 
            string("___"),
            string("+++") ) << endl;

    cout << endl;
};
