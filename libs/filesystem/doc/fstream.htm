boost/filesystem/fstream.hpp

Introduction

The C++ Standard Library's <fstream> header uses const char* to pass arguments representing file names, with that usage occurring in seven times.

The Filesystem Library's fstream.hpp header provides, in namespace boost::filesystem, equivalent components, except that the seven  const char* arguments have been replaced by const path& arguments.

The implementation simply uses the <fstream> standard library components as base classes, and then overrides the members with changed argument types.

For the moment, the only other documentation beyond the tutorial and examples is the header itself.  See boost/filesystem/fstream.hpp.
