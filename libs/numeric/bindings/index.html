Bindings documentation

Description

The bindings library contains bindings to external (numeric) libraries (that is: extenal to boost) such as BLAS, LAPACK, CBLAS and AMOS. These bindings can be used for uBlas vectors and matrices, std::vector, boost::array etc.

Traits

The traits module contains traits classes (what else;) for obtaining a pointer to the storage, the size, stride and leading dimension of a vector and matrix. The file traits.hpp contains a default trait that can be used. For applying these traits however on e.g. ublas matrices, you also need to include ublas_matrix.hpp.

Amos

Blas

First the blas-bindings are split in plain C-bindings for blas and next higher-level bindings. The C-bindings are direct matches on the fortran signature. The C++ bindings however also change the signature (e.g. return value not as extra parameter, ...)

cblas

LAPACK

Uses a traits class to map the functions for the different value_type's. This is possible for all calls for which the signature does not change for the different value_type's.
