Detailed Description

template<typename Geometry, bool IsConst>
struct ggl::vertex_iterator< Geometry, IsConst >

Meta-function which defines vertex_iterator type.

The vertex_iterator meta-function enables approaching any geometry with the same type of iterator. It is used within the library in conjuction with sections. A section defines a part of a geometry (linestring, polygon: outer ring or inner ring). The library, at that point, does not need to know if it is a polygon, multi-polygon or ring. Using this meta-function it still does not need to know that.

Template Parameters:
Geometry the geometry type
is_const,: true if const iterator is defined, else false
