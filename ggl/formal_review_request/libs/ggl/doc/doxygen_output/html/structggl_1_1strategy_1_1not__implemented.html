Detailed Description

Indicate compiler/library user that strategy is not implemented.

The strategy_traits class define strategies for point types or for point type combinations. If there is no implementation for that specific point type, or point type combination, the calculation cannot be done. To indicate this, this not_implemented class is used as a typedef stub.
