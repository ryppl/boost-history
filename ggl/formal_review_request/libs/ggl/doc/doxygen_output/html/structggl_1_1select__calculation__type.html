Detailed Description

template<typename Point1, typename Point2, typename CalculationType>
struct ggl::select_calculation_type< Point1, Point2, CalculationType >

Utility selecting the "calculation" type.

Based on two input point types, and an input calculation type, (which usually defaults to void), this meta-function selects the most appropriate:

  • if calculation type is specified, that one is used,
  • if it is void, the most precise of the two points is used
