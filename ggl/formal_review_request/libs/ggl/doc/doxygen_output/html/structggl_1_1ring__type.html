Detailed Description

template<typename Geometry>
struct ggl::ring_type< Geometry >

Meta-function which defines ring type of (multi)polygon geometry.

a polygon contains one exterior ring and zero or more interior rings (holes). The type of those rings is assumed to be equal. This meta function retrieves the type of such rings.
