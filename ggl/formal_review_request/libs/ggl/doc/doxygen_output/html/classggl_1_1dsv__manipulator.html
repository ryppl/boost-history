Detailed Description

template<typename Geometry>
class ggl::dsv_manipulator< Geometry >

Generic geometry template manipulator class, takes corresponding output class from traits class.

Stream manipulator, streams geometry classes as DSV streams

Example:
Small example showing how to use the dsv class 
 


Note:
the template parameter must be specified. If that is inconvient, users might use streamdsv which streams geometries as manipulators, or the object generator make_dsv
