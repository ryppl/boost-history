Detailed Description

Source descriptions:
  • OGC description: The mathematical centroid for this Surface as a Point. The result is not guaranteed to be on this Surface.
  • From Wikipedia: Informally, it is the "average" of all points
See also:
http://en.wikipedia.org/wiki/Centroid
Note:
The "centroid" functions are taking a non const reference to the centroid. The "make_centroid" functions return the centroid, the type has to be specified.

There are versions where a centroid calculation strategy can be specified

Geometries:
  • RING:
  • BOX: the centroid of a 2D or 3D box is the center of the box
  • POLYGON
  • POINT, LINESTRING, SEGMENT: trying to calculate the centroid will result in a compilation error

Calculate centroid.

The function centroid calculates the centroid of a geometry using the default strategy. Polygon should be closed, and can be orientated either way

Parameters:
geometry a geometry (e.g. closed ring or polygon)
c reference to point which will contain the centroid
Exceptions:
centroid_exception if calculation is not successful, e.g. because polygon didn't contain points
Example:
Example showing centroid calculation 
{
    ggl::polygon<ggl::point_xy<double> > poly;
    ggl::read_wkt("POLYGON((0 0,0 7,4 2,2 0,0 0))", poly);
    // Center of polygon might have different type then points of polygon
    ggl::point_xy<float> center;
    ggl::centroid(poly, center);
    std::cout << "Centroid: " << center.x() << "," << center.y() << std::endl;
}
