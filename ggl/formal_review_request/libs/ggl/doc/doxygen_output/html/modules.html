Modules

Here is a list of all modules:
  • access: get/set coordinate values, make objects, clear geometries, append point(s)
  • area calculation
  • arithmetic: arithmetic operations on points
  • boolean relationships (equals, disjoint, overlaps, etc)
  • buffer calculation
  • centroid calculation
  • combine: add a geometry to a bounding box
  • concept checking
  • convert geometries from one type to another
  • convex hull calculation
  • coordinate systems
  • core: meta-functions for geometry types
  • distance calculation
  • envelope calculation
  • intermediate calculation
  • length calculation
  • loops and for-each functionality
  • overlay operations (intersection, union, clipping)
  • perimeter calculation
  • sectionalize: split a geometry (polygon, linestring, etc)
  • selection: check if a geometry is "selected" by a point
  • simplification (generalization)
  • traits: adapt geometries
  • transformations
  • utility: utilities
  • x Extension svg: Stream SVG (Scalable Vector Graphics)
