Detailed Description

template<typename P>
struct ggl::segment< P >

Class segment: small containing two (templatized) point references.

From Wikipedia: In geometry, a line segment is a part of a line that is bounded by two distinct end points, and contains every point on the line between its end points.

Note:
The structure is like std::pair, and can often be used interchangeable. So points are public available. We cannot derive from std::pair<P&, P&> because of reference assignments. Points are not const and might be changed by the algorithm (used in intersection_linestring).
