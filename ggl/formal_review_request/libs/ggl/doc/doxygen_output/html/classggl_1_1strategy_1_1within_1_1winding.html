Detailed Description

template<typename P, typename PS = P>
class ggl::strategy::within::winding< P, PS >

Within detection using winding rule.

Template Parameters:
P point type of point to examine
PS point type of segments, defaults to P
Author:
Barend Gehrels
Note:
The implementation is inspired by terralib http://www.terralib.org (LGPL)

but totally revised afterwards, especially for cases on segments

More efficient (less comparisons and no divison) than the cross count algorithm

Only dependant on "side", -> agnostic, suitable for latlong
