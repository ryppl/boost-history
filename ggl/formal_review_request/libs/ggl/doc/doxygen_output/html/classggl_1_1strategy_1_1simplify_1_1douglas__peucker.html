Detailed Description

template<typename R, typename O_IT, typename PSDS>
class ggl::strategy::simplify::douglas_peucker< R, O_IT, PSDS >

Implements the simplify algorithm.

The douglas_peucker strategy simplifies a linestring, ring or vector of points using the well-known Douglas-Peucker algorithm. For the algorithm, see for example:

See also:
http://en.wikipedia.org/wiki/Ramer-Douglas-Peucker_algorithm

http://www2.dcs.hull.ac.uk/CISRG/projects/Royal-Inst/demos/dp.html

Template Parameters:
R boost range
O_IT output iterator
PSDS point-segment distance strategy to be used
Note:
This strategy uses itself a point-segment-distance strategy which can be specified
