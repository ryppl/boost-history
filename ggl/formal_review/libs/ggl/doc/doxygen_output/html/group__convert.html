Detailed Description

Convert from one geometry type to another type, for example from BOX to POLYGON
Geometries:
  • point to box -> a zero-area box of a point
  • box to ring -> a rectangular ring
  • box to polygon -> a rectangular polygon
  • ring to polygon -> polygon with an exterior ring (the input ring)
  • polygon to ring -> ring, interior rings (if any) are ignored
