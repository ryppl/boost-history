Struct template static_multiply<X, dim< T, V >>

boost::units::static_multiply<X, dim< T, V >> — Multipy a value_type and a dim.

Synopsis

template<typename T, typename V, typename X> 
struct static_multiply<X, dim< T, V >> {
  // types
  typedef dim< T, typename static_multiply< X, V >::type > type;
};

Description

static_multiply operations on dim are only defined between dim and value_type or value_type and dim. Operation delegates to value_type.
