Predicates

Overview

Predicates are included in <boost/string_algo/predicate.hpp> header.
The algorithms are resolving if a substring is contained in the input string under various conditions. These conditions are suffix and prefix check, equality check and containment check.

Description

The following table shows the summary of all predicates. Each algorithm returns a true if the condition is satisfied.

Function Description
starts_with Checks if the substring is a prefix of the input.
ends_with Checks if the substring is a suffix of the input.
contains Checks if the input contains an instance of substring.
equals Checks if both operands are equal.
( Note: this is two-directional equivalent of std::equal algorithm)

Synopsis

Iterator based variants.  

namespace boost {

    template< typename ForwardIterator1T, typename ForwardIterator2T >
    inline bool starts_with( 
        ForwardIterator1T Begin, 
        ForwardIterator1T End, 
        ForwardIterator2T SubBegin,
        ForwardIterator2T SubEnd );
    template< typename ForwardIterator1T, typename ForwardIterator2T >
    inline bool ends_with( 
        ForwardIterator1T Begin, 
        ForwardIterator1T End, 
        ForwardIterator2T SubBegin,
        ForwardIterator2T SubEnd );
    template< typename ForwardIterator1T, typename ForwardIterator2T >
    inline bool contains( 
        ForwardIterator1T Begin, 
        ForwardIterator1T End, 
        ForwardIterator2T SubBegin,
        ForwardIterator2T SubEnd );
    template< typename ForwardIterator1T, typename ForwardIterator2T >
    inline bool equals( 
        ForwardIterator1T Begin, 
        ForwardIterator1T End, 
        ForwardIterator2T SubBegin,
        ForwardIterator2T SubEnd );
}

Sequence variants. 

namespace boost {

    template< typename SeqT1, typename SeqT2 >
    inline bool starts_with( const SeqT1& Input, const SeqT2& Substr );
    template< typename SeqT1, typename SeqT2 >
    inline bool ends_with( const SeqT1& Input, const SeqT2& Substr );
    template< typename SeqT1, typename SeqT2 >
    inline bool contains( const SeqT1& Input, const SeqT2& Substr );
    template< typename SeqT1, typename SeqT2 >
    inline bool equals( const SeqT1& Input, const SeqT2& Substr );
}

Parameters:

typename ForwardIterator1T A forward iterator delimiting the input
typename ForwardIterator2T A forward iterator delimiting the search sequence
typename SeqT1, typename SeqT2 A sequence types (must satisfy the ContainerConcept)
ForwardIterator1T Begin The beginning of the input sequence
ForwardIterator1T End The end of the input sequence
ForwardIterator2T SubBegin The beginning of the substring sequence
ForwardIterator2T SubEnd The end of the substring sequence
const Seq1T& Input The input sequence
const Seq2T& Substr The substring sequence

Examples

Following example shows basic usage of predicate algorithms. More examples can be found in the library tests in <libs/string_algo/test/predicate_test.cpp>. 

#include <string>
#include <iostream>
#include <boost/string_algo/predicate.hpp>

using namespace std;
using namespace boost;

void predicate_example()
{
    cout << "* Predicate Example *" << endl << endl;

    string str1("123xxx321");
    string str2("abc");

    cout << "str1 starts with \"123\": " << 
        (starts_with( str1, string("123") )?"true":"false") << endl; 
    cout << "str1 ends with \"123\": " << 
        (ends_with( str1, string("123") )?"true":"false") << endl; 

    cout << "str1 contains \"xxx\": " << 
        (contains( str1, string("xxx") )?"true":"false") << endl; 

    // Note: equals have to be explicitly qualified, 
    // otherwise it conflicts with std::equals
    cout << "str2 equals \"abc\": " << 
        (boost::equals( str2, string("abc") )?"true":"false") << endl; 

    cout << endl;
}
