Interface IAnimal

The following pseudocode

interface IAnimal {
    void speak();
};

can be rendered in the Interface Definition Language (IDL) as follows:

BOOST_IDL_BEGIN(IAnimal)
    BOOST_IDL_FN0(speak, void)
BOOST_IDL_END(IAnimal)
