Description

The class access is used by the Interfaces library to access member data of user-defined classes without requiring the data to be declared public. The only situation in which users need to refer directly to the class access is in conjunction with the class template delegating. (See set_delegate.)

Headers

<boost/interfaces/access.hpp>

Synopsis

namespace boost { namespace interfaces { 

class access { ... };

} } // End namespace boost::interfaces
