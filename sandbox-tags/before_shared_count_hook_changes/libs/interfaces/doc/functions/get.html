Function Template get

Description
Headers
Synopsis

Description

The function template get returns a pointer through which the member functions of an object bound to a shared_obj or unique_obj may be accessed.

Headers

<boost/interfaces/shared_obj.hpp>
<boost/interfaces/unique_obj.hpp>

Reference

Synopsis

namespace boost { namespace interfaces {

template<typename Interface>
fixed_view<Interface>* get(const shared_obj<Interface>& obj);

template<typename Interface>
fixed_view<Interface>* get(const unique_obj<Interface>& obj);

} } // End namespace boost::interfaces
