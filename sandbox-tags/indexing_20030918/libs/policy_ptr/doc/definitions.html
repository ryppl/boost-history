Introduction

Smart pointer terminology is briefly explained, but MC++D is a more complete reference.

Definitions

Smart Pointer: A C++ object that simulates simple pointers by implementing operator-> and the unary operator*. [Alexandrescu 01] More generally, a smart pointer can be viewed as a generic RAII wrapper.

Policy: A policy defines a class interface or a class template interface...Policies have much in common with traits (Alexandrescu 2000a) but differ in that they put less emphasis on type and more emphasis on behaviour...Policy classes are not intended for standalone use; instead, they are inherited by, or contained within, other classes. [Alexandrescu 01] Simply put, a policy is a component of a generic type that contributes customizable behaviour.
