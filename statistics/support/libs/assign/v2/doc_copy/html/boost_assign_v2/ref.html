Convention

In addition to our previous conventions, code describing a feature is assumed to be preceded by 

using namespace ref;

Furthermore, in terms of our convened symbols, template argument U is either of T or T const. Finally, we denote ar either of the objects returned by csv_array() or array().

Interface

Header 

#include <boost/assign/v2/ref/array/interface.hpp>

Synopsis

The interface of ar, is akin to std::array<>, but it has, in addition, a rebind() member function.

Expression

Description 

ar.rebind( i, t )

Rebinds the ith reference to &t
