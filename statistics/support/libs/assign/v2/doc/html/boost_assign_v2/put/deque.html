csv_deque
deque
Header 

#include <boost/assign/v2/put/deque.hpp>

Description

The functions documented in this section, csv_deque() and deque() each constructs a container similar to std::deque<> whose interface for putting elements is shared with that of _csv_put and _csv_put, respectively.

csv_deque

Synopsis

Expression

Result type

csv_deque( t[0], ..., t[n-1] )

result_of::deque<T>::type

Example 

boost::for_each(
    as2::csv_deque<std::string>("x", "y", "z" ),
    os << bl::_1 << ' '
); // outputs x, y, z
