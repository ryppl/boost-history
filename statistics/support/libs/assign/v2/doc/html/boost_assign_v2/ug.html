Getting started

The library's file are in the directory structure rooted at boost/assign/v2. To be able to use all of the library's functionality, except macros, 

#include <boost/assign/v2.hpp>
using namespace boost::assign::v2;

To use only a subset of it, find the relevant section in the user's guide and include only the associated header.

Tip

Check portability, and if necessary ask for help.
