This section describes the usage of the tools of this library. The code examples given therein are in directory libs/assign/v2/example. For specific details, check the reference section.

Notation

Table 1.1. Symbols

Symbol

Description

cont

A container

s

A key

t

A data

u

A key data pair

arg

An argument to a function

obj

An object defined in this library

p

A parameter for mutating an object


To represent the types of the above, we typically use the a capital letter. For example T is the type of t.
