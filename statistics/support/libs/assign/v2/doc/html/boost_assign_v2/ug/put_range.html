Adaptor _put_range

Header 

#include <boost/assign/v2/put/range.hpp>

Description

A function that copies a range to container, which is particularly relevant if the latter does not have a contructor taking a range.

Synopsis

Expression

Result

to | adaptor::_put_range( from )

A reference to to

Example

In this example we show how to put a std::vector<>'s element into a std::queue<>. 

typedef std::vector<T> from_; from_ from( 3 );
from[0] = x; from[1] = y; from[2] = z;
typedef std::queue<T> to_; to_ to;
using namespace adaptor;
BOOST_ASSERT( (to | _put_range( from ) ).front() == x );
BOOST_ASSERT( to.back() == z );

Reversing the roles std::vector<> and std::queue<> in the above operation will not compile, because the input container, from, must model a forward range.
