logical or

Synopsis 

template< 
      typename T1
    , typename T2
    , typename T3 = false_t?
    ...
    , typename Tn = false_t?
    >
struct logical_or
{
    static bool const value = T1::value || T2::value || T3::value || ... Tn::value;
};

Description

logical_or class template performs short-circuit logical or (||) operation on its arguments.

Header 

#include "boost/mpl/logical/or.hpp"

Template parameters

 Parameters  Description  
T1, .., T5The logical_or's input parameters. Must be a model of [Intergal Expression]?.

Members

 Member  Description  
valueThe result of the logical or operation performed.

Example

Find the position of a type in a type_list? [1].

Notes

[1] This is an example of how to use the short-circuit logic operations to simplify your code; it is not the recommended way of searching for something in a type_list? or any other Sequence?. The find or find_if? algorithms offer a better way of doing it.
