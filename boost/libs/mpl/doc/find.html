find

Synopsis 

template<typename InputSequence, typename T>
struct find
{
    typedef /* implementation defined */ type;
};

Description

find algorithm returns the first iterator i in the range [Sequence::begin, Sequence::end) such that typeof(i::element_type) == typeof(T). If no such iterator exists then typeof(iterator)==typeof(Sequence::end).

Example 

typedef mpl::type_list?<char, int, unsigned, long, unsigned_long> list;
BOOST_STATIC_ASSERT(mpl::find<list, unsigned>::iterator::position == 2);
