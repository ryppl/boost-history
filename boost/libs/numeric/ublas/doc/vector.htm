Vector

Description

The templated class vector<T, F, A> is the base container adaptor for dense vectors.

Example

int main () {
    numerics::vector<double> v (3);
    for (int i = 0; i < v.size (); ++ i)
        v (i) = i;
    std::cout << v << std::endl;
}

Definition

Defined in the header vector.h.

Template parameters

Parameter Description Default
T The type of object stored in the vector.  
F Functor describing the storage organization. forward
A The type of the adapted array. unbounded_array<T>

Vector.
