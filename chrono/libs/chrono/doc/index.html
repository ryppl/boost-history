Introduction

The Boost Chrono library provides an implementation of the C++0x Standard Library's new time utilities and compile-time rational arithmetic. The implementation will work with most C++03 conforming compilers.

Examples

  • await_keystroke.cpp - A tiny program that times how long until a key is struck.
  • time2_demo.cpp - Howard Hinnant's original demonstration program.

Reference material

The most authoritative reference material for the library is the C++ Standards Committee's current Working Paper (WP). Found on the WG21 web site in the mailings section.

See sections:

  • 20.9 Time utilities [time]
  • 20.4 Compile-time rational arithmetic [ratio]
  • 20.6.7 Other transformations [meta.trans.other]

The C++ Standards Committee proposal for the library, N2661 - A Foundation to Sleep On, is very informative and provides motivation for key design decisions.
