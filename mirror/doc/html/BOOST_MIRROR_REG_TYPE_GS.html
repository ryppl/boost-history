Macro BOOST_MIRROR_REG_TYPE_GS

BOOST_MIRROR_REG_TYPE_GS — This macro is used to register a type, that is defined in the global scope, with Mirror. After successful registration the type can be reflected. This macro is equivalent to the BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE macro.

Synopsis

BOOST_MIRROR_REG_TYPE_GS(TYPE)

Description

The BOOST_MIRROR_REG_TYPE_GS macro registers a global scope type with the Mirror library.

To register a type defined in a named namespace use the BOOST_MIRROR_REG_TYPE. To register a typedefined type, use one of the BOOST_MIRROR_REG_TYPEDEF_GLOBAL_SCOPE, BOOST_MIRROR_REG_TYPEDEF_GS or BOOST_MIRROR_REG_TYPEDEF macros. To register a type which is defined inside of a class use either the BOOST_MIRROR_REG_TYPE_EMBEDDED or the BOOST_MIRROR_REG_TYPE_EMB macro.

Note
The registering must be done in the ::boost::mirror namespace.

The only argument is the name of the type to be registered.

To register a type foo defined in the global scope, 

struct foo { };

do this:

// note that the registering must be done
// in the boost::mirror namespace

namespace boost {
namespace mirror {

BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE(foo)
// or use the shorter version
BOOST_MIRROR_REG_TYPE_GS(foo)

} //namespace mirror
} //namespace boost

Afterwards the type can be reflected with the BOOST_MIRRORED_TYPE macro:

BOOST_MPL_ASSERT((
	reflects_type< BOOST_MIRRORED_TYPE(foo) >
));
BOOST_MPL_ASSERT((
	reflects_namespace< BOOST_MIRRORED_TYPE(foo)::scope >
));
BOOST_MPL_ASSERT((
	reflects_global_scope< BOOST_MIRRORED_TYPE(foo)::scope >
));

The Mirror library has the native C++ types pre registered:

namespace boost {
namespace mirror {

BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE(bool)
BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE(char)
BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE(wchar_t)
// ...
BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE(double)

} //namespace mirror
} //namespace boost
