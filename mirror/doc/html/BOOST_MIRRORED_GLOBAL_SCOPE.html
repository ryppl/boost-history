Macro BOOST_MIRRORED_GLOBAL_SCOPE

BOOST_MIRRORED_GLOBAL_SCOPE — This macro is used to reflect the global scope and expands into a specialization of the meta_namespace template, which provides meta-data about the global scope. To reflect a registered named namespace use the BOOST_MIRRORED_NAMESPACE.

Synopsis

BOOST_MIRRORED_GLOBAL_SCOPE()

Description

The BOOST_MIRRORED_GLOBAL_SCOPE() macro can be used to get the meta-data about the global scope, which is registered with the Mirror library by default.

Examples
BOOST_STATIC_ASSERT((
	reflects_global_scope< BOOST_MIRRORED_GLOBAL_SCOPE() >::value
));

BOOST_MPL_ASSERT((
	reflects_global_scope< BOOST_MIRRORED_GLOBAL_SCOPE() >
));

BOOST_MPL_ASSERT((
	::boost::is_same<
		BOOST_MIRRORED_NAMESPACE(::std)::scope,
		BOOST_MIRRORED_GLOBAL_SCOPE() 
	>
));

BOOST_MPL_ASSERT((
	::boost::is_same<
		BOOST_MIRRORED_TYPE(int)::scope,
		BOOST_MIRRORED_GLOBAL_SCOPE() 
	>
));

BOOST_MPL_ASSERT((
	::boost::is_same<
		BOOST_MIRRORED_TYPE(::std::wstring)::scope::scope,
		BOOST_MIRRORED_GLOBAL_SCOPE() 
	>
));
