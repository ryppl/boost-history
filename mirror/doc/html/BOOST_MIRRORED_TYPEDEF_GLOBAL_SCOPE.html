Macro BOOST_MIRRORED_TYPEDEF_GLOBAL_SCOPE

BOOST_MIRRORED_TYPEDEF_GLOBAL_SCOPE — This macro is used to reflect a typedef from the global scope, and expands into a specialization of the meta_type template, which provides meta-data about the reflected type. To reflect a typedef from a namespace use the BOOST_MIRRORED_TYPEDEF macro. To reflect a base type (not a typedef-ined) use the BOOST_MIRRORED_TYPE macro. To reflect the type of an expression use the BOOST_MIRRORED_TYPEOF macro.

Synopsis

BOOST_MIRRORED_TYPEDEF_GLOBAL_SCOPE(TYPEDEF)

Description

The BOOST_MIRRORED_TYPEDEF_GLOBAL_SCOPE macro reflects a type that is registered with the Mirror library as a global-scope typedef-ined type.

This macro is equivalent to BOOST_MIRRORED_TYPEDEF_GS. See the documentation of BOOST_MIRRORED_TYPEDEF_GS for more details.
