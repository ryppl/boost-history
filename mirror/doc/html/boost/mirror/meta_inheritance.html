Class template meta_inheritance

boost::mirror::meta_inheritance — This class template reflects the inheritance of one individual base class of a class. The meta_inheritance template should not be instantiated directly, MetaInheritances are obtained only as results of introspection operations on meta_base_classes.

Synopsis

template<typename unspecified> 
class meta_inheritance {
public:
  // types
  typedef mpl::int_<unspecified> position;
  typedef unspecified base_class;
  typedef meta_class< base_class > meta_base_class;
};

Description

This template allows to get compile-time meta-data about the inheritance of a base class of a class.

meta_inheritance public types

  1. typedef mpl::int_<unspecified> position;

    This member is an instantiation of the ::boost::mpl::int_<N> template and specifies the position of the meta_inheritance in the container and thus also the order of this base class in the list of base classes.

  2. typedef unspecified base_class;

    The base class type.

  3. typedef meta_class< base_class > meta_base_class;

    The meta_class reflecting the inherited base class.
