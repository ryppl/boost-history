Class template meta_class

boost::mirror::meta_class — This class template provides meta-data about the reflected class. Although the specializations of meta_class template can be used directly to get the meta data about a particular class (or type), it is recomended to use the reflection macro BOOST_MIRRORED_CLASS(CLASS) instead, to get the proper specializations of meta_class.

Synopsis

template<typename Class> 
class meta_class : public meta_type<Class>
{
public:
  // types
  typedef meta_base_classes<unspecified> base_classes;
  typedef meta_class_attributes<unspecified> attributes;
  typedef meta_class_all_attributes<unspecified> all_attributes;
};

Description

meta_class public types

  1. typedef meta_base_classes<unspecified> base_classes;

    The list of meta_inheritance specializations reflecting the base classes of the class reflected by this meta_class.

  2. typedef meta_class_attributes<unspecified> attributes;

    The container of meta_class_attributes reflecting the "own" attributes of a class. These are the attributes that are defined directly in the class reflected by this meta_class, not including the attributes inherited from the base classes.

  3. typedef meta_class_all_attributes<unspecified> all_attributes;

    The container of meta_class_attributes reflecting all attributes of a class including the ones inherited from base classes.
