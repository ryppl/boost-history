Class template meta_class

boost::mirror::meta_class — This class template is to provides meta-data about the reflected class. Although the specializations of meta_class template can be used directly to get the meta data about a particular class (or type), it is recomended to use the reflection macro BOOST_MIRRORED_CLASS(CLASS) instead, to get the proper specializations of meta_class.

Synopsis

template<typename Class> 
class meta_class : public meta_type<Class>   // The basic type meta-information about the reflected class.
					
{
public:
  // types
  typedef unspecified base_classes;
  typedef unspecified attributes;
  typedef unspecified all_attributes;
};

Description

meta_class public types

  1. typedef unspecified base_classes;

  2. typedef unspecified attributes;

  3. typedef unspecified all_attributes;
