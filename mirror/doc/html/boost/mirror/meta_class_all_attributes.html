Class template meta_class_all_attributes

boost::mirror::meta_class_all_attributes — This class template contains the meta_class_attributes reflecting the individual attributes of a class. More precisely, all attributes including the inherited ones. To inspect only the "own" attributes of a class, not including the inherited ones, use the specializations of the meta_class_attributes template. template. The meta_class_all_attributes template should not be instantiated directly, meta_class_all_attributes are obtained only as result of introspection operations on meta_class. The individual meta_class_attributes and iterators pointing to them should be obtained as results of the meta-object-sequence algoritms. See the examples for more details on usage of meta_class_all_attributes.

Synopsis

template<typename unspecified> 
class meta_class_all_attributes {
public:
  // types
  typedef meta_class<unspecified> scope;
};

Description

Example - Basic usage

meta_class_all_attributes public types

  1. typedef meta_class<unspecified> scope;

    Meta-class describing the scope where the reflected attributes are defined. The scope is the class that the reflected attributes belong to.
