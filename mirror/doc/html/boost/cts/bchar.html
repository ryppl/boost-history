Type definition bchar

bchar — This type is defined as wchar_t or as char based on the BOOST_CTS_USE_WIDE_CHARS preprocessing symbol being defined.

Synopsis

typedef char-or-wchar_t bchar;

Description

The bchar type is defined as follows: 

namespace boost {
  namespace cts {	
#ifdef BOOST_CTS_USE_WIDE_CHARS
    typedef wchar_t bchar;
#else
    typedef char bchar;
#endif
  } // namespace cts
} // namespace boost
