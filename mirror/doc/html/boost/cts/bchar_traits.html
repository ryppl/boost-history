Type definition bchar_traits

bchar_traits — The ::std::char_traits for the bchar character type.

Synopsis

typedef ::std::char_traits<bchar> bchar_traits;

Description

The bchar_traits type is defined as abbreviation for ::std::char_traits<bchar>.
