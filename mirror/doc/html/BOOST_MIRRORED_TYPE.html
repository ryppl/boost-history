Macro BOOST_MIRRORED_TYPE

BOOST_MIRRORED_TYPE — This macro is used to reflect a type and expands into a specialization of the meta_type template, which provides meta-data about the reflected type. To reflect a typedef-ined type use the BOOST_MIRRORED_TYPEDEF macro. To reflect the type of an expression use the BOOST_MIRRORED_TYPEOF macro.

Synopsis

BOOST_MIRRORED_TYPE(TYPE)

Description

The BOOST_MIRRORED_TYPE macro reflects a type previously registered with the Mirror library or a type derived from a registered type. The argument of this macro is a type.

Examples
Basic usage
BOOST_MIRRORED_TYPE(int)::base_name();

BOOST_MPL_ASSERT((
	reflects_global_scope<
		BOOST_MIRRORED_TYPE(int)::scope
	>
));

BOOST_MPL_ASSERT_NOT((
	reflects_global_scope<
		BOOST_MIRRORED_TYPE(::std::string)::scope
	>
));

BOOST_MIRRORED_TYPE(::std::wstring)::scope::full_name();
