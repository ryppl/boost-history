Macro BOOST_MIRROR_REG_TYPEDEF_GLOBAL_SCOPE

BOOST_MIRROR_REG_TYPEDEF_GLOBAL_SCOPE — This macro is used to register a type, that is defined in the global scope, with Mirror. After successful registration the type can be reflected. This macro is equivalent to the BOOST_MIRROR_REG_TYPEDEF_GS macro.

Synopsis

BOOST_MIRROR_REG_TYPEDEF_GLOBAL_SCOPE(TYPEDEF)

Description

See the documentation for the BOOST_MIRROR_REG_TYPEDEF_GS macro for more details.
