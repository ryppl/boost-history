Macro BOOST_CTS_USE_WIDE_CHARS

BOOST_CTS_USE_WIDE_CHARS — This preprocessor symbol is used to switch the definition of bchar and everything derived from this type.

Synopsis

BOOST_CTS_USE_WIDE_CHARS

Description

If the BOOST_CTS_USE_WIDE_CHARS symbol is defined then the type bchar is defined as wchar_t, otherwise it's defined as char: 

#include <boost/char_type_switch/string.hpp>

#ifdef BOOST_CTS_USE_WIDE_CHARS
BOOST_MPL_ASSERT((::boost::is_same< ::boost::cts::bchar, wchar_t >));
#else
BOOST_MPL_ASSERT((::boost::is_same< ::boost::cts::bchar, char >));
#endif
