Macro BOOST_MIRROR_REG_TYPE_EMB

BOOST_MIRROR_REG_TYPE_EMB — This macro is used to register a type, that is defined inside of a class. After successful registration the type can be reflected. This macro is equivalent to the BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE macro.

Synopsis

BOOST_MIRROR_REG_TYPE_EMB(WRAPPER, TYPE)

Description

The BOOST_MIRROR_REG_TYPE_EMB macro registers a type nested inside of a class with the Mirror library.

To register a type defined in a named namespace use the BOOST_MIRROR_REG_TYPE. To register a typedefined type, use one of the BOOST_MIRROR_REG_TYPEDEF_GLOBAL_SCOPE, BOOST_MIRROR_REG_TYPEDEF_GS or BOOST_MIRROR_REG_TYPEDEF macros. To register a type which is defined inside of a class use either the BOOST_MIRROR_REG_TYPE_EMBEDDED or the BOOST_MIRROR_REG_TYPE_EMB macro.

Note
The registering must be done in the ::boost::mirror namespace.

The first argument WRAPPER is the fully qualified name of the class inside of which the registered type is nested. The second argument TYPE is the typename of the type being registered.

The following code listings show how to register types "embedded" inside of other classes: 


// type (struct) bar is defined in the class foo
// that is defined in the global scope
struct foo 
{
  struct bar { };
};

// here the wrapping class foo is defined 
// in the namespace baz
namespace baz {
  struct foo 
  {
    struct bar { };
  };
} // namespace baz

// note that the registering must be done
// in the boost::mirror namespace

namespace boost {
namespace mirror {

// register the namespace first
BOOST_MIRROR_REG_NAMESPACE((baz))
// register the wrapping classes
BOOST_MIRROR_REG_TYPE_GLOBAL_SCOPE(foo)
BOOST_MIRROR_REG_TYPE(::baz, foo)
// register the embedded types (structs)
BOOST_MIRROR_REG_TYPE_EMB(::foo, bar)
// use the longer version on the second 
BOOST_MIRROR_REG_TYPE_EMBEDDED(::baz::foo, bar)

} //namespace mirror
} //namespace boost

Afterwards the types can be reflected with the BOOST_MIRRORED_TYPE macro:

BOOST_MPL_ASSERT((
  reflects_type< BOOST_MIRRORED_TYPE(foo::bar) >
));
BOOST_MPL_ASSERT((
  reflects_type< BOOST_MIRRORED_TYPE(baz::foo::bar) >
));
BOOST_MPL_ASSERT((
  reflects_class< BOOST_MIRRORED_TYPE(foo::bar)::scope >
));
BOOST_MPL_ASSERT((
  reflects_namespace< BOOST_MIRRORED_TYPE(foo::bar)::scope::scope >
));
BOOST_MPL_ASSERT((
  reflects_global_scope< BOOST_MIRRORED_TYPE(foo::bar)::scope::scope >
));
BOOST_MPL_ASSERT((
  reflects_class< BOOST_MIRRORED_TYPE(bar::foo::bar)::scope >
));
BOOST_MPL_ASSERT((
  reflects_namespace< BOOST_MIRRORED_TYPE(bar::foo::bar)::scope::scope >
));
BOOST_MPL_ASSERT_NOT((
  reflects_global_scope< BOOST_MIRRORED_TYPE(bar::foo::bar)::scope::scope >
));
BOOST_MPL_ASSERT((
  reflects_global_scope< 
    BOOST_MIRRORED_TYPE(bar::foo::bar)::scope::scope::scope 
  >
));
