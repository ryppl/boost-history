Macro BOOST_MIRRORED_TYPE_NS

BOOST_MIRRORED_TYPE_NS — This macro is a variation of the BOOST_MIRRORED_TYPE macro, which takes the name of the type split into two arguments; the full namespace name and the base type name.

Synopsis

BOOST_MIRRORED_TYPE_NS(FULL_NAMESPACE_NAME, TYPE)

Description

BOOST_MIRRORED_TYPE_NS(NAMESPACE, TYPE) is equivalent to BOOST_MIRRORED_TYPE(NAMESPACE :: TYPE).

Examples
BOOST_MIRRORED_TYPE_NS(::std, string)::base_name();

BOOST_MPL_ASSERT_NOT((
	reflects_global_scope<
		BOOST_MIRRORED_TYPE_NS(::std, string)::scope
	>
));

assert(
	BOOST_MIRRORED_TYPE(::std::wstring)::full_name() ==
	BOOST_MIRRORED_TYPE_NS(::std, wstring)::full_name()
);

assert(
	BOOST_MIRRORED_TYPEDEF(::boost::cts, bstring)::full_name() !=
	BOOST_MIRRORED_TYPE_NS(::boost::cts, bstring)::full_name()
);
