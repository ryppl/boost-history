Macro BOOST_MIRRORED_TYPEOF

BOOST_MIRRORED_TYPEOF — This macro is used to reflect the type of an expression.

Synopsis

BOOST_MIRRORED_TYPEOF(EXPRESSION)

Description

This macro is basically just a shorter form of 
	BOOST_MIRRORED_TYPE(BOOST_TYPEOF(EXPRESSION))
Examples
	BOOST_MIRRORED_TYPEOF(1+1)::base_name();

BOOST_MPL_ASSERT((
	reflects_global_scope<
		BOOST_MIRRORED_TYPEOF(1.0+1.0)::scope
	>
));
