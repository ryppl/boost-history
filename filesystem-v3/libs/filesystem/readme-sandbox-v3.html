Filesystem Version 3 in the Sandbox

To experiment with filesystem.v3:

  • Subversion export trunk into a temporary directory: http://svn.boost.org/svn/boost/trunk
  • From the trunk temporary, delete temporary-root/boost/filesystem and temporary-root/libs/filesystem
  • Subversion checkout filesystem-v3 from the sandbox: http://svn.boost.org/svn/boost/sandbox/filesystem-v3
  • Move temporary-root/* to the filesystem-v3-root just checked out.
