Boost Filesystem Version 3 Download and Installation

The alpha and beta releases of Boost.Filesystem are distributed as .bz2 and .zip files, differing only in having Unix or Windows line endings, respectively.

Copy Boost installation

The installation procedure overwrites existing Boost.Filesystem files, so you probably want to make a copy of your Boost installation to experiment with. Let's assume you copy your current boost_1_42_0 directory tree to directory boost-v3-test.

Download

  • Go to the Boost Vault with your web browser.
  • Click on the Filesystem-V3 Folder.
  • For the distribution file matching your operating system's line endings, click on the download icon (with the down arrow) in the Actions column.

Windows installation

  • Unzip the distribution file you just downloaded. On my Windows system, that produced a directory named d:\download\filesystem-v3-alpha-0.3
     
  • Change to the unziped distribution directory, and run the install_distro script with dot and the test directory as arguments. On my system, these are the command lines:
>cd D:\download\filesystem-v3-alpha-0.3
>install_distro . d:\boost-v3-test

All other operating systems installation

  • Unpack the distribution file you just downloaded. On my Linux system, that produced a directory named ~/filesystem-v3-alpha-0.3
     
  • Change to the unpacked distribution directory, and run the install_distro script with dot and the test directory as arguments. On my system, these are the command lines:
cd ~/filesystem-v3-alpha-0.3
$ ./install_distro.sh . ~/boost-v3-test
