Chain

Adaptor
Operator
Header

boost/assign/v2/chain.hpp

Note

This header does not include boost/assign/v2/chain/operator.hpp.

Description

A variant of join that caters to reference arrays

Adaptor

Header

boost/assign/v2/chain/adaptor.hpp

Description

Range adaptor syntax.

Synopsis 

namespace chain_aux{

    template<typename U>
    struct adaptor1;

    struct adaptor2;

    template<typename R1, typename U2>
    typename result_of::chain<R1, U2>::type
    operator|(R1 & r1, chain_aux::adaptor1<U2> const & h);

    template<typename R1, typename U2>
    typename result_of::chain<R1 const, U2>::type
    operator|(R1 const & r1, chain_aux::adaptor1<U2> const & h);

}// chain_aux

    const chain_aux::adaptor2 _chain;

namespace result_of{

    using chain_aux::result_of::chain;

}// result_of

Semantics

Notation

chained

result of r1 | _chain( r2 )

Expressions

Expression

Value

*( i + begin( chained ) )

*( i < size( r1 ) ? i + begin( r1 ) : i - size( r1 ) + begin( r2 ) )

Although a reference array is (generally) generated in place, its elements may be convertible to non-const lvalue references. Chaining preserves this property.

Examples

Example 1 

typedef std::string T;
std::vector<T> word;

boost::copy(
    as2::csv_deque( "O", "R" ) | as2::_chain( as2::csv_deque( "B", "I", "T" ) ),
    std::back_inserter( word )
);

BOOST_ASSIGN_V2_CHECK(
    boost::range::equal( word, as2::csv_deque<T>( "O", "R", "B", "I", "T" ) )
);

Example 2 

typedef int T; boost::array<T, 3> head; std::list<T> tail( 2 );

boost::copy(
    as2::csv_deque( 1, 2, 3, 4, 5 ),
    boost::begin( head | as2::_chain( tail ) )
);

BOOST_ASSIGN_V2_CHECK(
    boost::range::equal( head, as2::csv_deque( 1, 2, 3 ) )
);
BOOST_ASSIGN_V2_CHECK(
    boost::range::equal( tail, as2::csv_deque( 4, 5 ) )
);

Operator

Header

boost/assign/v2/chain/operator.hpp

Description

Equivalent semantics, different syntax, using operator&&. While convenient, it is controversial. Therefore, no header other than the one shown under Header defines this alternative syntax.

Semantics

Expression

Description

r1 && r2

Equivalent to r1 | _chain( r2 )

Example

Example 1 

std::vector<int> source( 8 ); boost::iota(source, 1);
boost::array<int, 4> head; int t, a, i, l;

using namespace boost::assign::v2;
boost::copy(
    source,
    boost::begin(
        head && (as2::ref::csv_array( t, a, i, l ) | as2::ref::_get )
    )
);

BOOST_ASSIGN_V2_CHECK(
    boost::range::equal( head, as2::csv_deque( 1, 2, 3, 4 ) )
);
BOOST_ASSIGN_V2_CHECK( t == 5 );
BOOST_ASSIGN_V2_CHECK( a == 6 );
BOOST_ASSIGN_V2_CHECK( i == 7 );
BOOST_ASSIGN_V2_CHECK( l == 8 );
