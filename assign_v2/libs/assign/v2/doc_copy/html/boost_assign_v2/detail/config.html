Synopsis

Constant

Default

Description

BOOST_ASSIGN_V2_LIMIT_LVALUE_CONST_ARITY

2

A functor's max # of arguments (k) for which any mixture of lvalue and const-lvalues is preserved

Beyond this limit, arguments are treated as either all lvalues or all const-lvalues.

Note

Applies to C++03, not C++0x
