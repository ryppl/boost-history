Side effect

put-implementation( arg[0], ..., arg[k - 1] )

A reference to put-implementation

modifier.impl(cont, f( arg[0], ..., arg[k - 1] ) )

Since the result is a reference to itself, put-implementation can be called successively: 

put-implementation( arg[0][0], ..., arg[k[0] - 1] )...( arg[0][n - 1], ..., arg[k[n - 1] - 1] )

Notice that the number of arguments is allowed to vary throughout the i = 0, ..., n - 1 sequence of calls. Due to this functor property, any file defining a class that derives from put_aux::crtp<> is named functor.hpp, and similarly for directories.
