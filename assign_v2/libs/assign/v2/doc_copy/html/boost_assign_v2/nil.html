_nil

Header 

#include <boost/assign/v2/detail/detail/keyword/nil.hpp>

Synopsis

Expression

Result type

_nil

implementation-defined

Description

If f designates some container generating function, f( _nil ) requests that the returned container be empty. [3]


[3] In particular, f() usually returns a container of size one, whose element is default initialized.
