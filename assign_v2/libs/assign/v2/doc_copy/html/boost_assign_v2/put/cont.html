#include <boost/assign/v2/put/container.hpp>

Description

An interface for modifying a container.

Functor

Table 1.10. Synopsis

Expression

Description

put( cont )

A functor deriving from crtp<> , with cont as the destination container


Example 

typedef int i_; typedef double d_;
typedef boost::tuple<d_&, i_> T;
d_ w = 2.7,  x = 2.71, y = 2.718, z = 0.0; std::vector<T> cont;
as2::put( cont )( w, 1 )( x, 2 )( y, 3 )( z );
BOOST_ASSIGN_V2_CHECK( &get<0>( cont[1] ) == &x );
BOOST_ASSIGN_V2_CHECK( get<1>( cont[1] ) == 2 );
T t3( z );
BOOST_ASSIGN_V2_CHECK( &get<0>( cont[3] ) == &z );
BOOST_ASSIGN_V2_CHECK( get<1>( cont[3] ) == get<1>( t3 ) );

Range

Header 

#include <boost/assign/v2/put/container/range.hpp>

Synopsis

Let t[0]...t[n] denote the dereferenced values of range r.

Expression

Result

Side effect 

put_range( cont, r );

A reference to cont 

put( cont )( t[0] )...( t[n] )


put_range<C>( r );

A container of type C

Calls put_range( cont, r ), with cont as the container to be returned

Example 

typedef int T; T x = 1, y = 2, z = 0;
std::vector<T> r( 3 ); r[0] = x; r[1] = y; r[2] = z;
typedef std::stack<T> lifo_;
BOOST_ASSIGN_V2_CHECK( as2::put_range<lifo_>( r ).top() == z );
