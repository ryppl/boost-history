Deduce

Fun
Modifier_tag
Header 

#include <boost/assign/v2/put/deduce.hpp>

Description

Default implementation of particular components. In constrast, section Modulo is the interface for overriding these default choices.

Tip

If the input is unary, and storage is by value, it might be more efficient to override the default with the identity functor.
