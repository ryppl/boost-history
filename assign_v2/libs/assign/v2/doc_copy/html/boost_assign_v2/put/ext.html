Header 

#include <boost/assign/v2/put/ext.hpp>

Description

Defines specializations of put_aux::modifier<> that are not captured by Std.

Iterate

Header 

#include <boost/assign/v2/put/ext/iterate.hpp>

Modulo keyword

_iterate

Synopsis

Expression

Implementation

_iterate = n

cont[ i++ ] = t

where i is initialized to n.

Example 

boost::array<int, 3> cont;
( as2::put( cont ) % as2::_iterate )( 1 )( 2 )( 3 );
BOOST_ASSIGN_V2_CHECK( cont[0] == 1 );
BOOST_ASSIGN_V2_CHECK( cont[2] == 3 );

Repeat

Header 

#include <boost/assign/v2/put/ext/repeat.hpp>

Modulo keyword

_repeat

Synopsis

Expression

Implementation

_repeat = n

m = n; while(--m) put-implementation

where put-implementation is the implementation prior to passing the parameter.

Example 

std::vector<int> cont;
( as2::put( cont ) % ( as2::_repeat = 2  ) )( -1 )( 0 )( 1 );
BOOST_ASSIGN_V2_CHECK( cont.size() == 6 );
BOOST_ASSIGN_V2_CHECK( cont.front() == -1 );
BOOST_ASSIGN_V2_CHECK( cont.back() == 1 );

Lookup

Header 

#include <boost/assign/v2/put/ext/lookup.hpp>

Modulo keyword

_lookup

Synopsis

Let key denote an Associative container's key.

Expression

Implementation

_lookup = f

cont[ key ] = f( cont[ key ] )

Example 

std::map<std::string, int> cont;
cont["jan"] = 31; cont["feb"] = 29; cont["mar"] = 32;
(
    as2::put( cont ) % ( as2::_lookup = ( lambda::_1 -= 1 ) )
)( "feb" )( "mar" );
BOOST_ASSIGN_V2_CHECK( cont["feb"] == 28 );
BOOST_ASSIGN_V2_CHECK( cont["mar"] == 31 );

Customization

Header 

#include <boost/assign/v2/put/ext/xxx.hpp>

This header has to be explicitly included

Procedure

The procedure for creating a keyword _param such that 

put-implementation % ( _param = param_arg )

conforms to the Modulo framework is obtained by following the procedure below. In each step below, the user is required to write a piece of code within namespace boost::assign::v2. The notation typename..., here, is not a variadic template parameter. It means that the choice of template parameters are unspecified.

  1. Define 
    template<typename Arg>
struct param
{
	// Constructor
    implementation-defined arg()const; 
};
  2. Define 
    struct param_keyword
{
	template<typename Arg>
    param<Arg> operator=(Arg const&); 
};
    where Arg is the type of param_arg.
  3. Define 
    namespace modifier_tag{

   	template<typename ...> struct param{};

}
  4. Define 
    namespace put_parameter{

   	template<typename Arg>
   	struct param{

           template<typename OldFun, typename OldTag>
           struct apply; // must derive from put_parameter::traits<>

   	};

}
  5. Specialize class modifier<modifier_tag::param<Tag> > such that its constructor takes as argument the result of ( _param = param_arg ).arg()
  6. Expand 
    BOOST_ASSIGN_V2_PUT_MODIFIER_EXT_XXX(param)
