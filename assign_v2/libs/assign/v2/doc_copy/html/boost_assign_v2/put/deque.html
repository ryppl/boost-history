Deque

Header 

#include <boost/assign/v2/put/deque.hpp>

Synopsis

Expression

Value

result_of::deque<T>::type

Container sharing with std::deque<T> the same accessors and complexity guarantees. In place of the usual modifiers (e.g. push_back()), a functor specified by put_aux::crtp<>.

Invokation

Let put-implementation denote the result result_of::deque<T>::type() i.e. an empty container.

Expression

Result type 

deque<T>( _nil )


put-implementation


deque<T>( arg[0], ..., arg[k - 1] )


put-implementation( arg[0], ..., arg[k - 1] )

Example 

typedef int T; T x = 1, z = 0;
as2::result_of::deque<T>::type cont = as2::deque<T>( x )()( z );
BOOST_ASSIGN_V2_CHECK( cont.front() == x );
BOOST_ASSIGN_V2_CHECK( cont[1] == T() );
BOOST_ASSIGN_V2_CHECK( cont.back() == z );

k = 1, 0, 1, respectively
