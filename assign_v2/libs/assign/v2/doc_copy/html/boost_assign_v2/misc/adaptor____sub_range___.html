Adaptor __subrange_

Header 

#include <boost/assign/v2/misc/sub_range.hpp>

Description

Wraps a container library (such as returned by __deque__) with extra capabilities, such as relational operators.

Synopsis

Expression

Result type

__sub_range( r )

[tpl_boost_itr boost::iterator_range<>

]

Example 

std::vector<int> v(3, -1);  v[2] = 1;
   assert( as2::sub_range( v ) < as2::csv_deque( -1, 0, 1 )  );


os << as2::sub_range( 
	as2::ref::csv_array( "x", "y", "z" ) 
); //outputs xyz
