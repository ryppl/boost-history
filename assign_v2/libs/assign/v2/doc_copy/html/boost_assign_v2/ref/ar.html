Interface

Header 

#include <boost/assign/v2/ref/array/interface.hpp>

Synopsis

The interface of the containers in this section are akin to boost::array<>, but it has, in addition, a rebind() member function.

Expression

Description 

ar.rebind( i, t )

Rebinds the ith reference to &t

csv_array

Header 

#include <boost/assign/v2/ref/array/csv.hpp>

Synopsis

Expression

Result type

csv_array<U>( _nil )

nth_result_of::csv_array<0, U>::type

csv_array( t[0], ..., t[n-1] )

nth_result_of::csv_array<n, U>::type

In the second expression, U == T const if and only if t[i] is passed as a const lvalue or an rvalue, for at least one i, and U == T otherwise.

Example 

boost::for_each(
    as2::ref::csv_array( "x", "y", "z" ),
    os << bl::_1 << ' '
); //outputs x y z


std::string x, y, z;
as2::ref::csv_array( x, y, z ).assign( "w" );
assert( y == "w" );

array

Header 

#include <boost/assign/v2/ref/array/functor.hpp>

Synopsis

Expression

Result type

array( _nil )

nth_result_of::array<0, U>::type

array( t[0] ) ...( t[n-1] )

nth_result_of::csv_array<n, U>::type

The determination of U for the second expression is identical to that for csv_array().

Example 

boost::for_each(
    as2::ref::array( -1 )( 0 )( 1 ),
    os << bl::_1 << ' '
); // outputs -1 0 1
