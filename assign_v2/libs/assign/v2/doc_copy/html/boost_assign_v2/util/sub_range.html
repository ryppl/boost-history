sub_range

Header 

#include <boost/assign/v2/utility/sub_range.hpp>

Description

Fits a container with some functionality such as relational operators, or operator<<.

Synopsis

Expression

Result type

sub_range( cont )

boost::iterator_range<>

Example 

std::vector<int> v(3, -1);  v[2] = 1;
   assert( as2::sub_range( v ) < as2::csv_deque( -1, 0, 1 )  );


os << as2::sub_range( 
	as2::ref::csv_array( "x", "y", "z" ) 
); //outputs xyz
