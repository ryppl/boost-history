Range_get

Header 

#include <boost/assign/v2/ref/wrapper/range_get.hpp>

Synopsis

Let r denote a range of reference-wrappers, and it its iterator.

Table 1.14. Usage

Expression

Result

range_get( r )

Maps *it to it->get()

r | _get

range_get( r )


Example 

typedef int T; T x, y, z;
std::vector<T> r( 3 ); r[0] = 1; r[1] = 2; r[2] = 0;
boost::copy( r, boost::begin(
        as2::ref::csv_array( x, y, z ) | as2::ref::_get
    )
);
BOOST_ASSIGN_V2_CHECK( x == r[0] );
BOOST_ASSIGN_V2_CHECK( z == r[2] );
