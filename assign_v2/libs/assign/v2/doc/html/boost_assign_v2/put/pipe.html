Description

A variant of the Container that allows for modying a container in place.

Csv

Header 

#include <boost/assign/v2/put/pipe/csv.hpp>

Synopsis 

implementation-defined  _csv_put;

Semantics

Default

Expression

Result

Side effect 

cont | _csv_put( t[0], ..., t[n - 1] )

Reference to cont 

put( cont )( t[0] ) ...( t[n - 1] )

Modulo 

cont | ( _csv_put % p )( t[0], ..., t[n - 1] )

Reference to cont 

( put( cont ) % p )( t[0] ) ...( t[n - 1] )

This extends without difficulty to an arbitrary number of modulo parameters. In other words, the mechanism is composable.

Example 

typedef int T; T x = 1, y = 2, z = 0; std::vector<T> cont;
BOOST_ASSIGN_V2_CHECK( 
	( 
    	cont | ( as2::_csv_put % ( as2::_fun = ( lambda::_1 + 1 ) )
	)( x, y, z ) ).front() == ( x + 1 )
);
BOOST_ASSIGN_V2_CHECK( cont.back() == ( z + 1 ) );

Functor

Header 

#include <boost/assign/v2/put/pipe/functor.hpp>

Synopsis 

implementation-defined  _put;

Semantics

Default

Expression

Result

Side effect 

cont | _put( arg[0], ..., arg[k - 1] )

Reference to cont 

put( cont )( arg[0], ..., arg[k - 1] )

In terms of our conventions, the above assumes n = 1, but it extends without difficulty to abitrary n.

Modulo

Expression

Result

Side effect 

cont | ( _put % p )( arg[0], ..., arg[k - 1] )

Reference to cont 

( put( cont ) % p )( arg[0], ..., arg[k - 1] )

This extends without difficulty to an arbitrary number of modulo parameters. In other words, the mechanism is composable.

Example 

typedef std::string T;
std::set<T> assoc;
T x = "isomer", y = "ephemeral", z = "prosaic";
BOOST_ASSIGN_V2_CHECK( 
	( assoc | as2::_put( x )( y )( z ) ).count( x ) == 1 
);
BOOST_ASSIGN_V2_CHECK( assoc.count( z ) == 1 );

Range

Header 

#include <boost/assign/v2/put/pipe/range.hpp>

Synopsis

Expression

Result 

cont | _put_range( r );


put_range( cont, r )


boost::type<C>() | _put_range( r );


put_range<C>( r )

Example 

typedef int T; T x = 1, y = 2, z = 0;
std::vector<T> r( 3 ); r[0] = x; r[1] = y; r[2] = z;
typedef std::stack<T> lifo_;
BOOST_ASSIGN_V2_CHECK(
    (
        ::boost::type<lifo_>() | as2::_put_range( r )
    ).top() == z
);
