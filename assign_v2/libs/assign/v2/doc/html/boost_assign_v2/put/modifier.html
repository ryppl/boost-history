Insert

Header 

#include <boost/assign/v2/put/modifier/insert.hpp>

Description

A Standard modifier

Semantics

Expression

Description

put-adapter % _insert

Replaces put-adapter's modifier-tag with modifier_tag::insert

Tip

Don't misconstrue the existence of this overload of operator% as necessary to specify the stated modifier. In particular, the default, for a particular put-adapter, is the most natural choice. See the section of that adapter and section Modifier.Deduce

Example 

std::set<std::string> letters; ( as2::put( letters ) % as2::_insert )( "d" )( "a" )( "c" )( "b" );

assert(letters.lower_bound( "a" ) == boost::begin( letters ) );
assert(letters.upper_bound( "d" ) == boost::end( letters ) );

Semantics

Notation

Instance of put_aux::adapter_modifier< modifier_tag::lookup<Arg> >

arg

Copy of argument passed as _lookup = arg

Adapter

Expression

Description

put-modifier( cont, key )

cont[ key ] = arg( cont[ key ] )

Meta

Expression

Description

mpl::apply2<put_aux::meta_modifier_tag<D>, keyword_lookup, Arg>::type

modifier_tag::lookup<Arg>

Pre-condition

Expression

Requirement

result-type of put-adapter's functor

convertible to C::key_type

Invocation

Expression

Description

put-adapter % ( _lookup = arg )

Replaces put-adapter's modifier-tag with modifier_tag::lookup<Arg>

The precondition is usually not met by default. In common cases, this suffices: 

put-adapter % ( _fun = lambda::_1 ) % ( _lookup = arg )

Example

Days-in-each-moth 

typedef std::map<std::string, int> C; C year; ( as2::put( year )( "feb", 28 )( "apr", 30 )( "jun", 30 )( "sep", 30 )( "nov", 30 ) 
    % ( as2::_fun = _1 ) %  ( as2::_lookup = ( _1 = 31 ) ) )( "jan" )( "mar" )( "may" )( "jul" )( "aug" )( "oct" )( "dec" );

assert(year["jan"] == 31 );
assert(year["dec"] == 31 );

Calls year["jan"] = 31 etc.
