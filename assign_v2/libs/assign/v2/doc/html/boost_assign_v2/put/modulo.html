Header 

#include <boost/assign/v2/put/modulo.hpp>

Description

An interface for overriding the default implementation. We add to our conventions that param denotes a user-specified parameter that is used only this framework.

Synopsis

Expression

Result

put-implementation % param

An implementation with a new type or state

There are two pre-defined values for param, the type of _fun and that of _modifier, which are described in the following sections.

Fun

Keyword 

_fun

Synopsis

Expression

Result

put-implementation % (_fun = f)

Replaces the internal functor of put-implementation by f

Tip

See section Functor for candidates for f

Example 

typedef int T; T x = 1, y = 2, z = 0;
std::vector<int> cont;
(
    as2::put( cont ) % ( as2::_fun = ( lambda::_1 + 1 ) )
)( x )( y )( z );
BOOST_ASSIGN_V2_CHECK( cont.front() == ( x + 1 ) );
BOOST_ASSIGN_V2_CHECK( cont.back() == ( z + 1 ) );

Modifier

Keyword 

_modifier

Synopsis

Expression

Result

put-implementation % (_modifier = tag() )

Replaces the internal modifier of put-implementation by an instance of put_aux::modifier<tag>

Note

It is customary, for each tag, to create a shortcut (called a keyword) for the right hand side of % in the above expression. See, for instance, section Std.

Example 

typedef int T; T x = 1, y = 2, z = 0;
std::deque<int> cont;
( as2::put( cont ) % ( as2::_modifier
    = as2::modifier_tag::push_front() ) )( x )( y )( z );
BOOST_ASSIGN_V2_CHECK( cont.front() == z );
BOOST_ASSIGN_V2_CHECK( cont.back() == x );
