converter

Header 

#include <boost/assign/v2/utility/convert.hpp>

Synopsis

Expression

Description

Implementation 

Cont cont = converter( r );

Copy constructs cont

Forwards to converter( r ).type<Cont>() 

converter( r ).type<Cont>();

Returns an object of type Cont

See rules.

Rules

The rules of conversion are a kind of switch statement, whose (only two) cases and the corresponding implementation are given below:

Condition

Implementation

Cont is an adapter or boost_array<> 

boost::type<Cont> | put_range( r )

True. 

Cont( boost::begin( r ), boost::end( r ) )

Example 

std::vector<int> v( 3 ); v[0] = -1; v[1] = 0; v[2] = 1;
std::stack<int> lifo = as2::converter( v ); 
assert( lifo.top() == 1 );


std::vector<int> v( 3 ); v[0] = -1; v[1] = 99; v[2] = 1;
typedef boost::array<int, 3> ar_;
assert( ( as2::converter( v ).type<ar_>() )[1] == 99 );
