Notation

cont

A container

C

cont's type

r

A range

args...

A list of arguments

a1, ..., ak

Individual elements of args...

t...

A list of values, each to be inserted into a container

T

A container's value-type

t1, ..., tn

Individual elements of t...

Convention

A reference array designates, in this library, an array of reference wrappers. See section Ref.Wrapper.

A keyword is an identifier with prefix _. Invoking it usually returns an object that specifies an operator overload.
