Include

Description

Overview of directory boost/assign/v2/include, which enables faster lookup of a particular feature. For example, 

#include <boost/assign/v2/include/csv_deque.hpp>

is equivalent to 

#include <boost/assign/v2/deque/csv_deque.hpp>
