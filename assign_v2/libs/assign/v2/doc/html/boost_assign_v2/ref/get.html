_get

Header 

#include <boost/assign/v2/ref/wrapper/get.hpp>

Synopsis

Let r denote a range of reference-wrappers.

Table 1.4. Usage

Expression

Result

r | _get

A range of type result_of::range_get<R>::type


Let it1 denote the iterator of r and it2 that of the transformed one.

Table 1.5. Property

Expression

Result

*it2

it1->get()


Comment

This transformation is necessary before passing a range of reference-wrappers to a write range algorithm.

Example 

std::vector<int> v( 3 ); v[0] = -1; v[1] = 0; v[2] = 1;
int x, y, z;
boost::copy(
    v,
    boost::begin(
        as2::ref::csv_array( x, y, z ) 
            | as2::ref::_get 
    )
);
assert( x == -1 );
assert( z == 1 );
