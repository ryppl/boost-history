Note

To be able to alter the references using a write algorithm, the reference-wrappers have to be mapped to references. This transformation can achieved using the range_get() functionality

Example 

typedef int const T;
T x = 1, y = 2, z = 0;
typedef as2::ref::nth_result_of::csv_array<3, T>::type ar_;
ar_ ar = as2::ref::csv_array( x, y, z );
{
    T& rx = ar[0]; T& rz = ar[2];
    BOOST_ASSIGN_V2_CHECK( &rx == &x );
    BOOST_ASSIGN_V2_CHECK( &rz == &z );
}
