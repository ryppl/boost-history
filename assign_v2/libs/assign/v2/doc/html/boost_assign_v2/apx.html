Containers

Known support for invoking put( cont )( args1... )...( argsn... ) is summarized below. Lack of support means that the library is unable to provide a default implementation, but the user may still be able to supply an explicit implementation : ( put( cont ) % __ked_option__ )( args1... )...( argsn... ).
