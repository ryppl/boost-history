Dispatch

Header 

#include <boost/assign/v2/utility/convert/dispatch.hpp>

Synopsis 

namespace convert_aux{

    template<typename C, typename R> C dispatch(R const& r, convert_tag::put);
    template<typename C, typename R> C dispatch(R const& r, convert_tag::copy);

}// convert_aux

Semantics

Expression

Description

dispatch<C>( r, convert_tag::put() )

Default constructs cont and modifies it through put_range( cont, r );

dispatch<C>( r, convert_tag::copy() )

Constructs cont by taking the begin and end iterators of r

dispatch<C>( r )

Forwards to the binary counterpart with convert_aux::deduce_tag<C, R>::type as second argument

Name-lookup macro

Description

A macro for bringing converter() within a user-specified namespace, such as to invoke name lookup.

Synopsis 

BOOST_ASSIGN_V2_CONVERTER( U, Seq )

Parameters

Symbol

Description

U

A name such as cont<A, B, C>

Seq

A sequence of template arguments such as (A)(B)(C)

Example 

std::vector<int> v( 3 ); v[0] = 1; v[1] = 2; v[2] = 0;
typedef boost::array<int, 3> ar_;
BOOST_ASSIGN_V2_CHECK(
    ( as2::converter( v ).type<ar_>() )[1] == v[1]
);


std::vector<int> v( 3 ); v[0] = -1; v[1] = 0; v[2] = 1;
std::stack<int> lifo = as2::converter( v );
BOOST_ASSIGN_V2_CHECK( lifo.top() == 1 );
