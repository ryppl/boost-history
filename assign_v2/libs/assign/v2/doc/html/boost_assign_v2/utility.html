Header 

#include <boost/assign/v2/utility/csv.hpp>

Description

Maps a sequence of values to unary functor calls over these values.

Synopsis

Let f denote a functor of type F, and T[0],..., T[n - 1] the types of t[0], ..., and t[n - 1], respectively.

Expression

Returns

csv( f, t[0], ..., t[n - 1] )

f( t[0] ) ... ( t[n - 1] )

Expression

Returns

result_of::csv<F const&, ::boost::mpl::vector<T[0],..., T[n - 1]>

The type of f( t[0] ) ... ( t[n - 1] )

]

This type is determined by recursive application of BOOST_TYPEOF_TPL.

Example 

typedef int T; T x = 1, y = 2, z = 0;
std::list<T> cont; 
as2::csv( as2::put( cont ) , x, y, z );
BOOST_ASSIGN_V2_CHECK( cont.front() == x );
BOOST_ASSIGN_V2_CHECK( cont.back() == z );
