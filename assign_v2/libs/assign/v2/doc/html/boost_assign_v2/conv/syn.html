Synopsis

In any code snippet under heading Synopsis, and more generally those that describe a feature, identifiers in the namespace of this library, boost::assign::v2, are not fully qualified. Another way to say this, is to assume that the code is preceded by the following using declaration. 

using namespace boost::assign::v2;

Code that describes a feature is expressed in terms of symbols and indexes, never actual data-types (such as int).
