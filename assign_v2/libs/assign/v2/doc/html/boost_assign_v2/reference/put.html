Description

A CRTP class template with a variadic functor interface that maps arguments to a value and inserts it in a container supplied by the derived class. Each of the mapping and the insertion mechanism, termed a modifier, are specified by template paramaters.

The semantics above are borrowed from range adaptors, but they differ in that the latter transforms a range, whereas, here, the state of a container is modified. In each case, however, composability follows, which is explained here

Example

Example 1 

typedef const char* T; typedef std::string str_; std::deque<T> cont;
assert( 
    boost::range::equal(
        cont | as2::_csv_put( "x", "y", "z" ),
        as2::csv_deque( "x" )( "y" )( "z" )
    )
);
assert( str_( cont[1] ) == "y" );
assert( str_( cont[2] ) == "z" );

"x", "y" and "z" are kept as const char(&)[2]

Example 2 

boost::array<int, 2> interval; boost::array<int, 6> all6;
BOOST_ASSIGN_V2_CHECK(
    boost::range::equal(
        all6 | as2::_put( 1 )( 2 )( as2::as_arg_list( interval | as2::_csv_put( 3, 4 ) ) )( 5 )( 6 ),
        as2::csv_deque( 1, 2, 3, 4, 5, 6 )
    )
);
BOOST_ASSIGN_V2_CHECK(
    boost::range::equal( interval, as2::csv_deque( 3, 4 ) )
);

Example 3 

std::deque<int> powers; int front = (
    powers | ( as2::_csv_put % as2::_push_front )( 16, 8, 4, 2, 1 )
).front();

assert( front == 1  );
assert( powers[4] == 16 );
