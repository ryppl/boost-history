Class template basic_scoped_loc

boost::persistent::basic_scoped_loc — Holds a locator with exclusive ownership.

Description

basic_loc::remove() is called as soon as this basic_scoped_loc is removed from the database.

Template parameters:

  • T The type of the object this locator can refer to

  • TxMgr The transaction manager used for accessing the object
