Description

basic_scoped_loc public construct/copy/destruct

  1. basic_scoped_loc();

    Constructs an empty basic_scoped_loc

    Throws: Nothing 

  2. basic_scoped_loc(T * p);

    Constructs a basic_scoped_loc to a new database object. Takes ownership of p. If an exception is thrown, delete p is called.

    • no_active_transaction

    Pointer to the new object.
