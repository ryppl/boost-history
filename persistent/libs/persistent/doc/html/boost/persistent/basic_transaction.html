Description

basic_transaction public construct/copy/destruct

  1. basic_transaction();

    Constructs a basic_transaction object If there already is an active transaction, the new transaction will be a nested transaction of the active transaction. Sets the active transaction to the new transaction.

    Throws:

    • no_active_database

    • finalize_error

    • io_failure

    • thread_resource_error 

  2. ~basic_transaction();

    Destructs the basic_transaction object The transaction is rolled back if it was not yet committed.

basic_transaction public member functions

  1. void commit() ;

    Commits the transaction. If this is a nested transaction, sets the active transaction to the parent transaction. If this is a root transaction, resets the active transaction.

    Throws:

    • isolation_exception This transaction conflicted with another transaction.

    • no_active_database

    • no_active_transaction

    • archive_exception

    • io_failure

    • thread_resource_error

    • Any exception thrown by the following user-supplied functions: T::T(), serialize(), reconstruct(), equal(), finalize() 

  2. void rollback() ;

    Unwinds all changes made during this transaction. If this is a nested transaction, sets the active transaction to the parent transaction. If this is a root transaction, resets the active transaction.

    Throws:

    • no_active_database

    • io_failure

    • thread_resource_error
