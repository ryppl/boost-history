  3. unspecified begin_transaction() ;
    Begins a new global transaction.

    Throws: Nothing

    Returns:

    An unspecified value that is convertible to transaction

    Notes:

    The return type can be transaction itself if it is CopyConstructible, a rvalue reference if it is MoveConstructible, or any other type that is convertible to transaction. 

  4. void commit_transaction(transaction & tx) ;
    Commits a global transaction.

    Commits all resource transactions that are associated with the global transaction tx. If there is more than one resource transaction, performs a two-phase commit.

    Throws: isolation_exception, archive_exception, io_failure, thread_resource_error, any exception thrown by user-supplied functions accessed persistent types

    Parameters:

    tx

    The global transaction 

  5. void rollback_transaction(transaction & tx) ;
    Rolls back a global transaction.

    Throws: io_failure, thread_resource_error

    Parameters:

    tx

    The global transaction
