Description

null_atomic_storage does nothing but forward the calls to a model of StorageEngine. When null_atomic_storage is used and the application is aborted, a corrupt database file is left with no way to recover it.

Only members that are not part of the AtomicStorageEngine concept are documented here.

Template parameters:

  • Storage A model of StorageEngine that is used to store the objects.

  • Threads true if this null_atomic_storage is accessed by multiple threads concurrently
