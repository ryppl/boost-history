Struct DataBlock

boost::persistent::DataBlock — Concept that provides random access to a block of data.

Synopsis

// In header: <boost/persistent/concept/DataBlock.hpp>


struct DataBlock {
  // types
  typedef unspecified size_type;  // Integral type representing the size of the block or an offset. 

  // public member functions
  void write(size_type, void const *, unspecified) ;
  void read(size_type, void *, unspecified) ;
  void flush() ;
  void sync() ;
};

Description

DataBlock public member functions

  1. void write(size_type offset, void const * data, unspecified size) ;
    writes size bytes of data found at data to offset offset

    Throws: io_failure

    Parameters:

    data

    offset

    size

    The number of bytes to write.

    Notes:

    the type of size is not specified, so MPL integral constants can be passed 

  2. void read(size_type offset, void * data, unspecified size) ;
    reads size bytes of data found at offset offset to data

    Throws: io_failure

    Parameters:

    data

    offset

    size

    The number of bytes to read.

    Notes:

    the type of size is not specified, so MPL integral constants can be passed 

  3. void flush() ;
    Flushes internal buffers.

    Throws: io_failure 

  4. void sync() ;
    Writes all flushed data to the physical storage device.

    May be called concurrently. Blocks until the data has reached the storage device.

    Does not flush internal buffers.

    Throws: io_failure, unsupported_exception
