Description

basic_loc is the equivalent locator to a C++ raw pointer. It does not implement any ownership management, persistent objects no longer in use need to removed using basic_loc::remove().

Template parameters:

  • T The type of the object this locator can refer to

  • TxMgr The transaction manager used for accessing the object
