Description

A locator of a database object FIXDOC loc does not implement any ownership management, it is the equivalent locator to a raw pointer. It comes with all the problems of raw pointers but extends these problems to the persistent space. It is recommended to use shared_loc, weak_loc, or scoped_loc instead whenever possible.
