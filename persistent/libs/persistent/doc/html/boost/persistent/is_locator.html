Struct template is_locator

boost::persistent::is_locator

Synopsis

// In header: <boost/persistent/basic_loc.hpp>

template<typename T> 
struct is_locator {
};

Description

A type trait that determines if T is a locator. If T is a type representing a locator, is_locator<T> derives from true_type, otherwise it derives from false_type. See Boost.TypeTraits documentation for information on true_type and false_type.
