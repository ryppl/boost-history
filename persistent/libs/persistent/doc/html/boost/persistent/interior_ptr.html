Description

An interior_ptr can contain a normal pointer or a pointer to a member of a persistent object. If it contains a pointer to a member of a persistent object, it keeps the persistent object in memory as long as it exists, to make sure the pointer is not invalidated.

    Notes:

    This constructor exists so that native pointers are convertible to interior_ptrs. It can not be used to create an interior_ptr to a member of a persistent object.
