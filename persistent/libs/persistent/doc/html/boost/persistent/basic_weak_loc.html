boost::persistent::basic_weak_loc — Stores a weak reference to an object managed by a basic_shared_loc.

Description

Template parameters:

  • T The type of the object this locator can refer to

  • TxMgr The transaction manager that is used to access the object

basic_weak_loc public construct/copy/destruct

  1. basic_weak_loc();
    Constructs an empty basic_weak_loc.

    Throws: Nothing 

  2. template<typename Y> basic_weak_loc(basic_shared_loc< Y, TxMgr > const & r);
    Constructs a basic_weak_loc that shares ownership with r and stores a copy of r.

    Y * must be convertible to T *.

    Throws: Nothing 

  3. basic_weak_loc(basic_weak_loc const & r);
    Constructs a basic_weak_loc that shares ownership with r and stores a copy of r.

    Throws: Nothing 

  4. template<typename Y> basic_weak_loc(basic_weak_loc< Y, TxMgr > const & r);
    Constructs a basic_weak_loc that shares ownership with r and stores a copy of r.

    Y * must be convertible to T *.

    Throws: Nothing 

  5. basic_weak_loc& operator=(basic_weak_loc const & r);
    Equivalent to weak_ptr(r).swap(*this). 
  6. template<typename Y> 
  basic_weak_loc& operator=(basic_weak_loc< Y, TxMgr > const & r);
    Equivalent to weak_ptr(r).swap(*this). 
  7. template<typename Y> 
  basic_weak_loc& operator=(basic_shared_loc< Y, TxMgr > const & r);
    Equivalent to weak_ptr(r).swap(*this).

basic_weak_loc public member functions

  1. bool expired() const;

    An object is reachable as long as there is at least one basic_shared_loc to the same object stored in the database, or the stored object is a new object created in this transaction.

    Note that this behaviour differs from boost::weak_ptr::expired()'s: If the last basic_shared_loc is removed in the current transaction the object may remain reachable through a basic_weak_loc until this transaction is committed.

    Throws: no_active_transaction_manager, isolation_exception io_failure, thread_resource_error

    Returns:

    true if the stored object is still reachable through this basic_weak_loc. 

  2. basic_shared_loc< T, TxMgr > lock() const;
    Equivalent to expired() ? basic_shared_loc<T,TxMgr>() : basic_shared_loc<T,TxMgr>(*this). 
  3. void reset() ;
    Equivalent to basic_weak_loc().swap(*this). 
  4. void swap(basic_weak_loc & b) ;
    Exchanges the contents of the two basic_weak_locs.

    Throws: Nothing 

  5. void swap(basic_weak_loc & a, basic_weak_loc & b) ;
    Equivalent to a.swap(b). 
  6. template<typename U> 
  bool operator<(basic_weak_loc< U, TxMgr > const & b) const;

    Throws: Nothing

    Returns:

    An unspecified value that provides strict weak ordering.
