Description

basic_weak_loc public construct/copy/destruct

  1. basic_weak_loc();

    Constructs an empty basic_weak_loc

    Throws: Nothing 

  2. template<typename Y> basic_weak_loc(basic_shared_loc< Y, TxMgr > const & r);

    Constructs a basic_weak_loc that shares ownership with r and stores a copy of r. Y * must be convertible to T *.

    Throws: Nothing 

  3. basic_weak_loc(basic_weak_loc const & r);

    Constructs a basic_weak_loc that shares ownership with r and stores a copy of r. Y * must be convertible to T *.

    Throws: Nothing 

  4. template<typename Y> basic_weak_loc(basic_weak_loc< Y, TxMgr > const & r);

    Constructs a basic_weak_loc that shares ownership with r and stores a copy of r. Y * must be convertible to T *.

    Throws: Nothing

  7. template<typename Y> 
  basic_weak_loc& operator=(basic_shared_loc< Y, TxMgr > const & r);

basic_weak_loc public member functions

  1. bool expired() const;

    Returns if the stored object is still reachable through this basic_weak_loc. An object is reachable as long as there is at least one basic_shared_loc to the same object stored in the database, or the stored object is a new object created in this transaction.

    Note that this behaviour differs from boost::weak_ptr::expired()'s: If the last basic_shared_loc is removed in the current transaction the object remains reachable through a basic_weak_loc until this transaction is committed.

    Throws:

    • no_active_database

    • isolation_exception

    • archive::archive_exception

    • io_failure

    • thread_resource_error

    • Any exception thrown by the following user-supplied functions: T::T(), serialize(), reconstruct() 

  2. basic_shared_loc< T, TxMgr > lock() const;
    Equivalent to expired() ? basic_shared_loc<T,TxMgr>() : basic_shared_loc<T,TxMgr>(*this). 
  3. void reset() ;
    Equivalent to basic_weak_loc().swap(*this). 
  4. void swap(basic_weak_loc & b) ;

    Exchanges the contents of the two basic_weak_locs.

    Throws: Nothing 

  5. void swap(basic_weak_loc & a, basic_weak_loc & b) ;
    Equivalent to a.swap(b). 
  6. template<typename U> 
  bool operator<(basic_weak_loc< U, TxMgr > const & b) const;

    Provides a strict weak ordering of basic_weak_locs

    Throws: Nothing

    Returns:

    An unspecified value that provides strict weak ordering.
