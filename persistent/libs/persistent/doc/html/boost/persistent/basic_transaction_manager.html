Description

Model of TransactionManager. Only members that are not part of that concept are documented here.

Template parameters:

  • Resources Either the type of the only resource manager used, or a MPL Sequence containing the types of the resource managers used.

  • Threads true if multiple threads are used to access this transaction manager.

  • TThreads true if multiple threads are used to access the same transaction. Can be false if multiple threads are used to access the transaction manager, but not to access the same transaction.
