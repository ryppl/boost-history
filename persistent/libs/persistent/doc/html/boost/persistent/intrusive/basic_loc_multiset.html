Description

Template parameters:

  • T The type this container can store locators to

  • TxMgr The transaction manager that is used to access the stored objects

  • ConstantTimeSize true if member size() should return in constant time. This reduces concurrency of threads.

  • Compare A BinaryPredicate that can less-than compare two objects of type T

  • Tag The tag of the basic_loc_set_hook that is used by this basic_loc_set

basic_loc_multiset public construct/copy/destruct

  1. basic_loc_multiset(Compare const & comp = Compare());
    Constructs an empty multiset.

    Complexity: Constant.

    Throws: If the copy constructor/operator() of comp throws. 

  2. template<typename Iterator> 
  basic_loc_multiset(Iterator begin, Iterator end, 
                     Compare const & comp = Compare());
    Constructs an empty multiset and inserts elements from [b, e).

    Complexity: Linear in N if [b, e) is already sorted using comp and otherwise N * log N, where N is the distance between first and last

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone(), exceptions thrown by the copy constructor/operator() of the value_compare object.

    Requires:

    Dereferencing iterator must yield an value of type value_type. cmp must be a comparison function that induces a strict weak ordering.

  18. template<typename Cloner, typename Disposer> 
  void clone_from(const basic_loc_multiset & src, Cloner cloner, 
                  Disposer disposer) ;
    Erases all the elements from *this.

    calls Disposer::operator()(pointer), clones all the elements from src calling Cloner::operator()(const_reference ) and inserts them on *this. Copies the predicate from the source container.

    If cloner throws, all cloned elements are unlinked and disposed calling Disposer::operator()(pointer).

    Complexity: Linear to erased plus inserted elements.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone(), exceptions thrown by the cloner or the predicate copy assignment.

    Requires:

    Disposer::operator()(pointer) shouldn't throw.

    Cloner should yield to nodes equivalent to the original nodes.

  22. iterator insert_before(const_iterator pos, reference value) ;
    Inserts x into the tree before "pos".

    Complexity: Constant time.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone()

    Requires:

    "pos" must be a valid iterator (or end) and must be the succesor of value once inserted according to the predicate

    Notes:

    This function does not check preconditions so if "pos" is not the successor of "value" tree ordering invariant will be broken. This is a low-level function to be used only for performance reasons by advanced users.

    Complexity: Average complexity is constant time.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone()

    Notes:

  56. void replace_node(iterator replace_this, reference with_this) ;
    Replaces replace_this in its position in the tree with with_this. The tree does not need to be rebalanced.

    Complexity: Constant.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone()

    Requires:

    replace_this must be a valid iterator of *this and with_this must not be inserted in any tree.

    Notes:

    This function will break container ordering invariants if with_this is not equivalent to *replace_this according to the ordering rules. This function is faster than erasing and inserting the node, since no rebalancing or comparison is needed.
