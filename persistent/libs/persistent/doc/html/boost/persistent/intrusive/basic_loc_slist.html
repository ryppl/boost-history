Description

Template parameters:

  • T The type this container can store locators to

  • TxMgr The transaction manager that is used to access the stored objects

  • ConstantTimeSize true if member size() should return in constant time. This reduces concurrency of threads.

  • Tag The tag of the basic_loc_slist_hook that is used by this basic_loc_slist

basic_loc_slist public construct/copy/destruct

    Complexity: Constant

    Throws: Nothing. 

  2. template<typename Iterator> basic_loc_slist(Iterator begin, Iterator end);
    Constructs a list equal to [first,last).

    Requires:

    Dereferencing iterator must yield an value of type value_type.

    Erases all the elements from *this.

  39. void splice_after(const_iterator prev, basic_loc_slist & x, 
                  const_iterator * last = 0) ;
    Transfers all the elements of list x to this list, after the the element pointed by prev.

    No destructors or copy constructors are called.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone()

    Complexity: Linear to the elements contained in x.

    Requires:

    prev is an iterator to an element or x.end()/x.before_begin() in x.

    Notes:

    Iterators of values obtained from list x now point to elements of this list. Iterators of this list are not invalidated.

    If the optional parameter "last" is provided, it will be assigned to the last spliced element or prev if x is empty. This iterator can be used as new "prev" iterator for a new splice_after call. that will splice new values after the previously spliced values.

    No destructors or copy constructors are called.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone()

    Complexity: Linear to the elements contained in x plus linear to the elements before it.

    Requires:

    Notes:

  48. void sort() ;
    This function removes all of x's elements and inserts them in order into *this.

    The merge is stable; that is, if an element from *this is equivalent to one from x, then the element from *this will precede the one from x.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone(), exceptions thrown by std::less<T>

    Complexity: This function is linear time: it performs at most size() + x.size() - 1 comparisons.

    Requires:

    p must be a comparison function that induces a strict weak ordering and both *this and x must be sorted according to that ordering

    The lists x and *this must be distinct.

    Notes:

    Iterators are not invalidated.
