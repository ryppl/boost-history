boost::persistent::intrusive::basic_loc_set_hook — A base class for persistent types that can be added to a basic_loc_set or basic_loc_multiset.

Description

Template parameters:

  • T The derived type that will be added to a basic_loc_set or basic_loc_multiset

  • TxMgr The transaction manager

  • Tag A tag to differentiate basic_loc_set_hooks from each other, so a object can be added to more than one intrusive set
