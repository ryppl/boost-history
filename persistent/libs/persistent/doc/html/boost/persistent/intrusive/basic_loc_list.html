Description

Template parameters:

  • T The type this container can store locators to

  • TxMgr The transaction manager that is used to access the stored objects

  • ConstantTimeSize true if member size() should return in constant time. This reduces concurrency of threads.

  • Tag The tag of the basic_loc_list_hook that is used by this basic_loc_list

  35. template<typename Cloner, typename Disposer> 
  void clone_from(const basic_loc_list & src, Cloner cloner, 
                  Disposer disposer) ;
    Erases all the elements from *this.

    calls Disposer::operator()(pointer), clones all the elements from src calling Cloner::operator()(const_reference) and inserts them on *this.

    If cloner throws, all cloned elements are unlinked and disposed calling Disposer::operator()(pointer).

    Complexity: Linear to erased plus inserted elements.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone(), exception thrown by the cloner

    Requires:

    Disposer::operator()(pointer) shouldn't throw. Cloner should yield to nodes equivalent to the original nodes.

  46. void merge(basic_loc_list & x) ;
    This function removes all of x's elements and inserts them in order into *this according to std::less<T>.

    The merge is stable; that is, if an element from *this is equivalent to one from x, then the element from *this will precede the one from x.

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, archive::archive_exception, finalize_error, io_failure, thread_resource_error, any exception thrown by the following user-supplied functions: T::T(), serialize(), load(), construct(), clone(), exceptions thrown by std::less<T>

    Complexity: This function is linear time: it performs at most size() + x.size() - 1 comparisons.

    Notes:

    Iterators are not invalidated
