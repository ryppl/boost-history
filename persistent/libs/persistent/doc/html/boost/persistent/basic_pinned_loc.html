Description

Constructing a basic_pinned_loc fills it with a locator to the object of type T that is pinned to Tag, if there is any.

basic_pinned_loc can be used to access the roots of an object graph.

Template parameters:

  • T The type of the object this locator can refer to

  • TxMgr The transaction manager used for accessing the object

  • Tag The type the object is pinned to

basic_pinned_loc public construct/copy/destruct

  1. basic_pinned_loc();
    If an object is pinned to Tag in the default resource, constructs a basic_pinned_loc to that object. Otherwise, constructs an empty basic_pinned_loc.

    Throws: no_active_transaction_manager, io_failure, thread_resource_error, unsupported_exception 

  2. basic_pinned_loc(basic_pinned_loc const & r);
    If an object is pinned to Tag in the resource referred to by ResourceTag, constructs a basic_pinned_loc to that object. Otherwise, constructs an empty basic_pinned_loc.

    Parameters:

    r

    Source locator

    Throws: Nothing

    Throws: no_active_transaction_manager, no_active_transaction, isolation_exception, finalize_error, io_failure, thread_resource_error
